# CODEXA Voice Audio Assets

Este diretório contém arquivos de áudio para easter eggs e feedback sonoro.

## Arquivos Esperados

| Arquivo | Trigger | Descrição |
|---------|---------|-----------|
| `should-i-stay-or-should-i-go.mp3` | "CODEXA ATIVAR" | Som de ativação (Stranger Things easter egg) |

## Como Adicionar

1. Coloque o arquivo MP3 neste diretório
2. O nome deve ser exatamente: `should-i-stay-or-should-i-go.mp3`
3. Teste com `/v` e diga "CODEXA ATIVAR"

## Fallback

Se o arquivo não existir, o sistema fará fallback para TTS:
> "Codexa ativado. Sistema operacional."
