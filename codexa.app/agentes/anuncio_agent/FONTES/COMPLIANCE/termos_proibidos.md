# Termos Proibidos - ANVISA, INMETRO, CONAR

**Version**: 1.0.0 | **Focus**: Compliance Regulatorio + Fallbacks

---

## Overview

Este arquivo lista termos que NAO podem ser usados em anuncios de marketplace, organizados por orgao regulador e categoria. Inclui alternativas seguras.

---

## 1. ANVISA - Agencia Nacional de Vigilancia Sanitaria

### Categorias Reguladas
- Alimentos e suplementos
- Cosmeticos e higiene pessoal
- Medicamentos
- Produtos para saude
- Saneantes

### Termos PROIBIDOS (Claims Terapeuticos)

| Termo Proibido | Alternativa Segura |
|----------------|-------------------|
| cura | auxilia, contribui |
| trata | ajuda a promover |
| previne doencas | favorece o bem-estar |
| elimina | reduz, minimiza |
| combate | auxilia no controle |
| anti-inflamatorio | com propriedades calmantes |
| antibacteriano | com acao higienizante |
| cicatrizante | que favorece a recuperacao |
| analgesico | que proporciona conforto |
| terapeutico | com beneficios funcionais |

### Claims de Saude PROIBIDOS

```
NUNCA USAR:
- "Cura ansiedade/depressao"
- "Trata diabetes/hipertensao"
- "Previne cancer/doencas cardiacas"
- "Elimina gordura/celulite"
- "Emagrece X kg em Y dias"
- "Substitui medicamentos"
- "Efeito garantido"
- "Aprovado por medicos" (sem comprovacao)
```

### Transformacoes Seguras

**ANTES (Proibido)**:
```
"Cha que CURA insonia e TRATA ansiedade"
```

**DEPOIS (Permitido)**:
```
"Cha que auxilia no relaxamento e contribui para noites mais tranquilas"
```

**ANTES (Proibido)**:
```
"Suplemento que ELIMINA gordura e EMAGRECE rapido"
```

**DEPOIS (Permitido)**:
```
"Suplemento que auxilia no metabolismo quando aliado a dieta equilibrada e exercicios"
```

### Palavras Seguras ANVISA

| Contexto | Termos Permitidos |
|----------|-------------------|
| Beneficio geral | auxilia, contribui, favorece, ajuda a promover |
| Bem-estar | proporciona bem-estar, promove equilibrio |
| Nutricao | fonte de nutrientes, rico em vitaminas |
| Energia | contribui para disposicao, fornece energia |
| Imunidade | auxilia nas defesas naturais do organismo |

## 2. INMETRO - Instituto Nacional de Metrologia

### Categorias Reguladas
- Eletroeletronicos
- Brinquedos
- Produtos infantis (cadeirinhas, bercos)
- Calcados
- Texteis
- Equipamentos de seguranca

### Termos PROIBIDOS

| Termo Proibido | Problema | Alternativa |
|----------------|----------|-------------|
| Certificado INMETRO (falso) | Fraude | Apenas se verdadeiro |
| Mais seguro que X | Comparacao | Atende normas de seguranca |
| 100% seguro | Absoluto | Testado conforme normas |
| Aprovado sem registro | Enganoso | Numero de registro visivel |

### Obrigatoriedades INMETRO

**Produtos que DEVEM ter selo**:
- Brinquedos
- Produtos infantis (cadeirinha auto, berco)
- Eletrodomesticos
- Fios e cabos eletricos
- Disjuntores
- Tomadas e plugues
- Capacetes (moto, ciclismo)
- Extintores

**Na Copy**:
```
CORRETO: "Produto com certificacao INMETRO - Registro XXXXX-X"
ERRADO: "Produto seguro" (sem mencionar certificacao)
```

### Especificacoes Tecnicas

**OBRIGATORIO informar com precisao**:
- Peso real (nao aproximado)
- Dimensoes exatas
- Capacidade correta
- Voltagem especifica
- Potencia verificada

**PROIBIDO**:
```
"Aproximadamente X kg" → Use valor exato
"Cerca de X litros" → Use capacidade verificada
"Mais ou menos X watts" → Use especificacao do fabricante
```

## 3. CONAR - Conselho de Autorregulamentacao Publicitaria

### Principios Gerais

A publicidade deve ser:
- Legal
- Decente
- Honesta
- Verdadeira

### Termos e Praticas PROIBIDAS

| Proibicao | Exemplo | Correcao |
|-----------|---------|----------|
| Comparacao depreciativa | "Melhor que marca X" | "Um dos melhores do mercado" |
| Superlativos absolutos | "O melhor do Brasil" | "Entre os mais vendidos" |
| Exploracao de medo | "Se nao comprar, vai sofrer" | Beneficios positivos |
| Apelo imperativo criancas | "Peca pra sua mae!" | Comunicacao para pais |
| Discriminacao | Qualquer tipo | Linguagem inclusiva |

### Claims de Posicionamento PROIBIDOS

```
NUNCA USAR:
- "#1 do mercado" (sem comprovacao)
- "Numero 1 em vendas" (sem fonte)
- "Melhor do Brasil/mundo"
- "Unico produto que..."
- "Lider absoluto"
- "Imbativel"
- "Insuperavel"
```

### Transformacoes Seguras

**ANTES (Proibido)**:
```
"O MELHOR smartphone do mercado, #1 em vendas"
```

**DEPOIS (Permitido)**:
```
"Um dos smartphones mais bem avaliados, com milhares de clientes satisfeitos"
```

**ANTES (Proibido)**:
```
"Unico produto APROVADO por especialistas"
```

**DEPOIS (Permitido)**:
```
"Produto reconhecido por sua qualidade comprovada em testes"
```

## 4. PROIBICOES GERAIS (Todos os Marketplaces)

### HTML e Formatacao

| Proibido | Razao |
|----------|-------|
| `<script>` | Seguranca |
| `<iframe>` | Seguranca |
| `<style>` | Padronizacao |
| JavaScript | Seguranca |
| CSS inline | Padronizacao |

### Emojis e Simbolos

| Proibido | Razao | Alternativa |
|----------|-------|-------------|
| Emojis | Profissionalismo | Texto descritivo |
| Unicode decorativo | Compatibilidade | Caracteres padrao |
| Simbolos especiais | Leitura | Pontuacao comum |

**Excecao**: Shopee permite emojis com moderacao.

### Links e Contatos

| Proibido | Razao |
|----------|-------|
| URLs externas | Desvio de trafico |
| Email | Contato fora plataforma |
| Telefone | Contato fora plataforma |
| WhatsApp | Desvio de venda |
| Redes sociais | Marketing externo |

### Precos e Promocoes

| Proibido | Razao | Onde Usar |
|----------|-------|-----------|
| Preco inline | Desatualizacao | Campo de preco |
| % desconto falso | Enganoso | Nunca |
| "Frete gratis" (falso) | Fraude | Apenas se real |
| "Promocao" (falsa) | Enganoso | Apenas em campanhas |

## 6. REGEX DE VALIDACAO

### Patterns para Detectar Termos Proibidos

```regex
# Claims terapeuticos
cura[rs]?|trata[mr]?|previne|elimina|combate

# Rankings absolutos
#1|n[uú]mero\s+1|melhor\s+do\s+(brasil|mundo)|[uú]nico\s+(no|do)

# Superlativos sem prova
o\s+mais\s+(barato|caro|eficaz)|o\s+melhor|totalmente\s+perfeito

# Links
https?://|www\.|\.com|\.br

# Precos inline
R\$\s*\d+[,.]\d+|por\s+apenas\s+\d+

# HTML
<[^>]+>|class\s*=|style\s*=
```

## 7. FALLBACK STRATEGIES

### Quando Detectar Termo Proibido

```
1. IDENTIFICAR → Qual termo e qual categoria
2. CLASSIFICAR → Severidade (CRITICAL, HIGH, MEDIUM)
3. SUBSTITUIR → Aplicar alternativa segura
4. VALIDAR → Verificar se copy mantem sentido
5. DOCUMENTAR → Registrar em NOTAS_DE_FALLBACK
```

### Niveis de Severidade

| Nivel | Acao | Exemplo |
|-------|------|---------|
| CRITICAL | Bloquear publicacao | Claims medicos sem ANVISA |
| HIGH | Substituir obrigatoriamente | "#1 do mercado" |
| MEDIUM | Substituir recomendado | Superlativos menores |
| LOW | Alertar apenas | Formatacao nao ideal |

### Template de Fallback

```markdown
NOTAS_DE_FALLBACK:
- Termo original: "[termo proibido]"
- Categoria: [ANVISA/INMETRO/CONAR/GERAL]
- Severidade: [CRITICAL/HIGH/MEDIUM/LOW]
- Substituicao aplicada: "[termo seguro]"
- Contexto: [onde foi encontrado]
- Validacao: [OK/REQUER REVISAO]
```

## 8. CHECKLIST DE COMPLIANCE

### Pre-Publicacao Obrigatoria

- [ ] Zero claims terapeuticos (ANVISA)
- [ ] Zero claims de posicionamento absoluto (CONAR)
- [ ] Certificacoes validas mencionadas (INMETRO)
- [ ] Zero links externos
- [ ] Zero informacoes de contato
- [ ] Zero precos inline
- [ ] Zero HTML proibido
- [ ] Zero emojis (exceto Shopee)
- [ ] Especificacoes tecnicas precisas
- [ ] Garantia informada corretamente

### Validacao Final

```
SE todos checks = OK → APROVADO para publicacao
SE algum check CRITICAL = FAIL → BLOQUEAR ate correcao
SE check HIGH/MEDIUM = FAIL → SUBSTITUIR e republiar
```
