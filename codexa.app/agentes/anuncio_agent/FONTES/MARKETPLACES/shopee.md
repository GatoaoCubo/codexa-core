# Shopee Brasil - Copy Patterns

**Version**: 1.0.0 | **Marketplace**: Shopee Brasil | **Focus**: Mobile-First + Engagement

---

## Overview

Shopee e mobile-first (90% dos usuarios). Copy aqui significa:
- **Titulos escaneiaveis** em telas pequenas
- **Imagens que vendem** (primeira impressao)
- **Participacao em campanhas** (Flash Sale, Cupons)

---

## 1. TITULOS (Max 120 chars)

### Formula Shopee
```
[PRODUTO] + [CARACTERISTICA 1] + [CARACTERISTICA 2] + [DIFERENCIAL] + [VARIACAO]
```

### Regras Especificas Shopee

| Regra | Correto | Errado |
|-------|---------|--------|
| Max 2x mesma keyword | `Blusa Feminina Cropped` | `Blusa Blusa Feminina Cropped Blusa` |
| Evitar spam | Texto natural | `PROMOÇÃO SUPER OFERTA DESCONTO` |
| Mobile-friendly | Informacao chave primeiro | Detalhes no inicio |
| Sem frete falso | Apenas se verdadeiro | "Frete Gratis" enganoso |

### Exemplos Otimizados Shopee

**Moda Feminina**:
```
Blusa Cropped Feminina Canelada Manga Curta Basica Cores Variadas
Vestido Midi Feminino Estampado Florido Verao Casual Elegante
Calca Jeans Feminina Skinny Cintura Alta Elastano Confortavel
```

**Acessorios**:
```
Relogio Digital Esportivo Masculino A Prova Dagua 50m LED
Bolsa Feminina Transversal Pequena Couro Sintetico Elegante
Oculos Sol Feminino Retro Vintage Proteção UV400 Luxo
```

**Eletronicos**:
```
Fone Ouvido Bluetooth 5.0 TWS Sem Fio Case Carregador IPX5
Carregador Portatil Powerbank 10000mAh USB-C Carga Rapida LED
Ring Light LED 26cm Tripé 2m Suporte Celular Youtuber Lives
```

### Diferenciais Shopee

**Palavras que performam**:
- `Pronta Entrega`
- `Envio Rapido`
- `Shopee Garantia`
- `Original` (quando verdadeiro)
- `Kit` / `Combo`
- `Promocao` (em campanhas)

## 2. DESCRICAO (Max 3,000 chars)

### Caracteristica Mobile

**90% dos usuarios leem no celular**. Isso significa:
- Paragrafos curtos (2-3 linhas)
- Listas com bullets
- Informacoes escaneiaveis
- CTA visivel

### Estrutura Shopee

```markdown
[TITULO ATRATIVO]
[Emoji opcional - mas com moderacao]

[BENEFICIO PRINCIPAL em 1 linha]

[DETALHES DO PRODUTO]
- Material: [especificacao]
- Tamanho: [medidas]
- Cor: [opcoes]
- Peso: [peso]

[POR QUE ESCOLHER ESTE PRODUTO?]
1. [Beneficio 1]
2. [Beneficio 2]
3. [Beneficio 3]

[ESPECIFICACOES TECNICAS]
[Lista formatada para leitura rapida]

[CONTEUDO DO PACOTE]
- 1x [item principal]
- [acessorios]

[DICAS DE USO / CUIDADOS]
[Informacoes uteis breves]

[GARANTIA]
[Politica de troca/devolucao]
```

### Formato de Texto

**SEM HTML** - Shopee nao aceita formatacao HTML.
Use apenas:
- Quebras de linha
- Hifens ou asteriscos para listas
- Espacamento entre secoes

## 3. IMAGENS (9 sempre!)

### Por Que 9 Imagens?

Shopee favorece anuncios com galeria completa:
- **Imagem 1**: Produto principal (fundo branco ou lifestyle)
- **Imagens 2-4**: Angulos diferentes
- **Imagens 5-6**: Detalhes/close-ups
- **Imagens 7-8**: Produto em uso
- **Imagem 9**: Especificacoes/tabela de medidas

### Especificacoes Tecnicas

| Aspecto | Requisito |
|---------|-----------|
| Dimensao minima | 800x800px |
| Dimensao recomendada | 1000x1000px |
| Formato | JPG, PNG |
| Tamanho max | 5MB |
| Texto em imagem | Minimo possivel |

### Restricoes de Imagem

- NAO usar watermarks de outros marketplaces
- NAO usar imagens de concorrentes
- NAO usar texto excessivo cobrindo produto
- NAO usar logos de marcas sem autorizacao

## 4. VIDEO (Aumenta 30% conversao!)

### Especificacoes Video Shopee

| Aspecto | Requisito |
|---------|-----------|
| Duracao | 15-60 segundos |
| Formato | MP4 |
| Orientacao | Vertical (9:16) preferido |
| Tamanho max | 30MB |

### Roteiro de Video Shopee

```
[0-3s] Hook visual - produto em destaque
[3-10s] Problema que resolve
[10-25s] Demonstracao de uso
[25-40s] Beneficios principais (3 max)
[40-55s] Variações/opcoes
[55-60s] CTA - "Adicione ao carrinho"
```

## 5. ESTRATEGIAS DE ENGAJAMENTO

### Participacao em Campanhas

| Campanha | Frequencia | Beneficio |
|----------|------------|-----------|
| Flash Sale | Diaria | Visibilidade maxima |
| Shopee Day (X.X) | Mensal | Trafico explosivo |
| Frete Gratis | Configuravel | Conversao aumentada |
| Cupom Loja | Sempre ativo | Fidelizacao |

### Live Commerce

Shopee prioriza vendedores que fazem lives:
- Agendamento previo
- Duracao minima 30 min
- Produtos em destaque
- Interacao com chat

### Metricas que Importam

| Metrica | Target | Impacto |
|---------|--------|---------|
| Taxa resposta | > 90% | Ranqueamento |
| Tempo resposta | < 12h | Conversao |
| Rating loja | > 4.8 | Confianca |
| Taxa envio no prazo | > 95% | Mercado Envios |

## 6. OTIMIZACAO MOBILE

### Checklist Mobile-First

- [ ] Titulo legivel em tela pequena (info chave primeiro)
- [ ] Primeira imagem mostra produto claramente
- [ ] Descricao em paragrafos curtos
- [ ] Especificacoes em formato lista
- [ ] Video vertical quando possivel
- [ ] CTAs claros e visiveis

### Hierarquia de Informacao

```
MAIS IMPORTANTE (topo)
├── Nome do produto
├── Preco / Desconto
├── Frete
└── Avaliacao

MEDIO IMPORTANTE (scroll)
├── Especificacoes principais
├── Variacoes
└── Descricao resumida

MENOS IMPORTANTE (final)
├── Detalhes tecnicos
├── Politicas
└── Informacoes do vendedor
```

---

## 7. ERROS COMUNS SHOPEE

| Erro | Consequencia | Correcao |
|------|--------------|----------|
| Keyword stuffing | Anuncio removido | Max 2x mesma palavra |
| < 9 imagens | Baixo ranqueamento | Sempre 9 fotos |
| Sem video | Perde 30% conversao | Adicionar video curto |
| Resposta lenta | Metricas ruins | Responder em < 12h |
| Frete falso | Suspensao | Apenas frete real |
| Reenvio sem corrigir | Ban permanente | Corrigir antes de reenviar |
