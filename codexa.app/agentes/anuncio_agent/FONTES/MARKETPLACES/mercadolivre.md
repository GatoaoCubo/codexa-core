# Mercado Livre - Copy Best Practices

**Version**: 1.0.0 | **Marketplace**: Mercado Livre Brasil | **Focus**: SEO + Conversion

---

## Overview

Mercado Livre e o maior marketplace da America Latina. Copy otimizada aqui significa:
- **SEO interno forte** (algoritmo proprio)
- **Confianca via Mercado Envio**
- **Resposta rapida a perguntas**

## 1. TITULOS (Max 60 chars)

### Formula de Ouro ML
```
[TIPO] + [CARACTERISTICA PRINCIPAL] + [DIFERENCIAL] + [ESPECIFICACAO]
```

### Regras Criticas

| Regra | Correto | Errado |
|-------|---------|--------|
| Zero conectores | `Whey Protein 1kg Isolado Chocolate` | `Whey Protein de 1kg com Chocolate` |
| Keywords primeiro | `Fone Bluetooth JBL Tune 510` | `JBL Tune 510 Fone Bluetooth` |
| Sem emojis | `Camiseta Dry Fit Academia` | `Camiseta Dry Fit Academia :fire:` |
| Sem HTML | Texto puro | `<b>Oferta</b>` |
| 58-60 caracteres | Maximo aproveitamento | Titulo curto desperdicado |

### Exemplos por Categoria

**Eletronicos**:
```
Fone Bluetooth JBL Tune 510BT Original Preto Dobravel Leve
Smartwatch Samsung Galaxy Watch 5 44mm GPS NFC Preto
Carregador Turbo USB-C 65W Universal Samsung Xiaomi iPhone
```

**Moda**:
```
Camiseta Dry Fit Masculina Academia Corrida Proteção UV50
Tenis Nike Air Max 270 Masculino Running Preto Original
Mochila Escolar Impermeavel Notebook 15 Polegadas USB Preta
```

**Casa e Decoracao**:
```
Organizador Gavetas Divisorias Ajustaveis Kit 8 Pecas Branco
Luminaria LED Mesa Escritorio Articulada Touch 3 Niveis
Panela Pressao Eletrica 6L Antiaderente Timer Programavel
```

### Keywords de Alta Conversao ML

**Trigger words que aumentam CTR**:
- `Original` (quando comprovavel)
- `Garantia` / `Garantia Estendida`
- `Pronta Entrega`
- `Full` (Mercado Envio Full)
- `Frete Gratis` (se aplicavel)

**Palavras de urgencia** (usar com moderacao):
- `Ultimas Unidades`
- `Lancamento`
- `Promocao` (apenas se verdadeiro)

## 2. DESCRICAO (Recomendado: 3,000-5,000 chars)

### Framework StoryBrand para ML

```
1. HEROI (comprador) tem um PROBLEMA
2. Encontra um GUIA (voce/produto)
3. Guia apresenta um PLANO
4. Chama para ACAO
5. Mostra SUCESSO se seguir
6. Mostra FRACASSO se nao seguir
7. TRANSFORMACAO final
```

### Estrutura Otimizada

```markdown
[ABERTURA - Dor do cliente]
Voce ja [problema comum]? Sabe como e [consequencia negativa]?

[APRESENTACAO - Solucao]
Apresentamos [produto] - a solucao definitiva para [problema].

[BENEFICIOS - 5-7 pontos]
- [Beneficio 1 com resultado tangivel]
- [Beneficio 2 com prova social]
- [Beneficio 3 com diferencial tecnico]
...

[ESPECIFICACOES TECNICAS]
- Material: [especificacao]
- Dimensoes: [medidas]
- Peso: [peso]
- Garantia: [periodo]

[CONTEUDO DA EMBALAGEM]
- 1x [produto principal]
- 1x [acessorio]
- Manual de instrucoes

[CTA - Call to Action]
Aproveite agora! Adicione ao carrinho e receba [beneficio de compra].

[GARANTIA/CONFIANCA]
Vendedor com [X] vendas e [X]% de avaliacoes positivas.
```

### HTML Permitido no ML

```html
<!-- Tags permitidas -->
<b>texto em negrito</b>
<i>texto em italico</i>
<u>texto sublinhado</u>
<br> <!-- quebra de linha -->
<p>paragrafo</p>

<!-- Tags PROIBIDAS -->
<script>, <iframe>, <style>, JavaScript, CSS inline
```

### Gatilhos Mentais para ML

| Gatilho | Exemplo de Uso |
|---------|----------------|
| Escassez | "Estoque limitado - ultimas X unidades" |
| Prova Social | "Mais de X clientes satisfeitos" |

| Reciprocidade | "Brinde exclusivo incluso" |
| Urgencia | "Oferta valida ate [data]" |

## 3. SEO INTERNO ML

### Fatores de Ranqueamento

| Fator | Peso | Como Otimizar |
|-------|------|---------------|
| Reputacao vendedor | 5-10% | Manter metricas verdes |
| Historico vendas | 10-15% | Volume + consistencia |
| Qualidade anuncio | 15-20% | Titulo, fotos, descricao completos |
| Preco competitivo | 10-15% | Monitorar concorrencia |
| Mercado Envio | 15-20% | Usar Full quando possivel |
| Taxa conversao | 10-15% | CTR + fechamento |

### Checklist SEO

- [ ] Titulo com 58-60 caracteres
- [ ] 8-10 keywords no titulo
- [ ] Zero conectores ("de", "para", "com", "e")
- [ ] Descricao >= 3,000 caracteres
- [ ] Todas as fotos em alta resolucao (min 800x800)
- [ ] EAN/GTIN preenchido (quando disponivel)
- [ ] Atributos tecnicos completos
- [ ] Categoria correta selecionada

## 4. PERGUNTAS E RESPOSTAS

### Templates de Resposta

**Pergunta sobre produto**:
```
Ola! Sim, [confirmacao]. [Detalhe adicional relevante].
Qualquer duvida, estamos a disposicao. Obrigado pelo interesse!
```

**Pergunta sobre frete**:
```
Ola! O frete e calculado pelo Mercado Envios baseado no seu CEP.
Apos inserir seu endereco, voce vera o valor e prazo exatos.
Trabalhamos com envio rapido! Obrigado!
```

**Pergunta sobre garantia**:
```
Ola! O produto possui garantia de [periodo] direto com o fabricante/vendedor.
Alem disso, voce tem a protecao da Compra Garantida do Mercado Livre.
Qualquer problema, resolvemos! Obrigado!
```

### Tempo de Resposta

- **Ideal**: < 1 hora
- **Aceitavel**: < 24 horas
- **Impacto negativo**: > 24 horas (afeta ranqueamento)

## 5. ERROS COMUNS A EVITAR

| Erro | Impacto | Solucao |
|------|---------|---------|
| Titulo curto demais | Perde SEO | Usar 58-60 chars |
| Conectores no titulo | Desperdiça espaco | Remover "de", "para", "com" |
| Fotos baixa qualidade | Baixa conversao | Min 800x800, fundo branco |
| Descricao generica | Perguntas excessivas | Detalhar especificacoes |
| Preco fora do mercado | Sem vendas | Pesquisar concorrencia |
| Resposta lenta | Perde ranqueamento | Responder em < 24h |
| Claims falsos | Suspensao | Apenas claims comprovados |
