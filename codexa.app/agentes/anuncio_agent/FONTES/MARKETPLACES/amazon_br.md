# Amazon Brasil - Copy Requirements

**Version**: 1.0.0 | **Marketplace**: Amazon Brasil | **Focus**: Style Guide Compliance + A+ Content

---

## Overview

Amazon e o marketplace mais rigoroso em padroes. Copy aqui significa:
- **Style Guide obrigatorio** (sem excecoes)
- **Backend Keywords** (SEO oculto)
- **A+ Content** (diferenciacao visual)

---

## 1. TITULOS (Max 200 chars)

### Formula Amazon (Style Guide)
```
[MARCA] + [LINHA/MODELO] + [MATERIAL/CARACTERISTICA CHAVE] + [COR] + [TAMANHO] + [QUANTIDADE]
```

### Regras do Style Guide

| Regra | Obrigatorio | Exemplo |
|-------|-------------|---------|
| Primeira letra maiuscula | Sim | `Samsung Galaxy Watch` |
| Marca primeiro | Sim | `JBL Tune 510BT...` |
| Sem ALL CAPS | Sim | NAO usar `PROMOÇÃO` |
| Sem simbolos | Sim | NAO usar `!@#$%` |
| Sem preco/promocao | Sim | NAO mencionar desconto |
| Sem "Amazon"/"Prime" | Sim | Nunca usar no titulo |
| Sem nome do seller | Sim | Nao incluir loja |

### Exemplos Conformes

**Eletronicos**:
```
Samsung Galaxy Watch 5 Smartwatch 44mm Bluetooth GPS NFC Preto
JBL Tune 510BT Fone de Ouvido Bluetooth Dobravel com Microfone Preto
Apple AirPods Pro 2a Geracao com Estojo de Carga MagSafe Branco
```

**Casa**:
```
Tramontina Jogo de Panelas Aluminio Antiaderente 5 Pecas Vermelho
Philips Walita Liquidificador 1200W 3L 12 Velocidades Preto
Electrolux Aspirador de Po Vertical Sem Fio 25.2V Cinza
```

**Moda**:

## 2. BULLET POINTS (5 obrigatorios)

### Estrutura de Bullet

```
[BENEFICIO/CARACTERISTICA em MAIUSCULA]: [Explicacao detalhada com beneficio para o cliente]
```

### Regras de Bullets

| Regra | Requisito |
|-------|-----------|
| Quantidade | 5 bullets (sempre) |
| Caracteres | 150-250 por bullet (max 500) |
| Inicio | Letra maiuscula |
| Final | Sem ponto se frase unica |
| Conteudo | Beneficios > features |
| Proibido | Precos, promocoes, shipping |

### Exemplo de 5 Bullets Otimizados

**Produto**: Fone Bluetooth

```
1. QUALIDADE DE SOM PREMIUM: Drivers de 40mm entregam graves profundos e agudos cristalinos para uma experiencia sonora imersiva em musicas, podcasts e chamadas

2. BATERIA DE LONGA DURACAO: Ate 40 horas de reproducao continua com uma unica carga, ideal para viagens longas e uso diario sem preocupacao com recarga

3. CONFORTO PARA USO PROLONGADO: Almofadas macias e ajuste ergonomico permitem horas de uso sem desconforto, perfeito para trabalho remoto e sessoes de estudo

4. CONECTIVIDADE VERSATIL: Bluetooth 5.0 com alcance de 10m e conexao multiponto permite parear dois dispositivos simultaneamente para alternar entre celular e notebook

5. DESIGN DOBRAVEL PORTATIL: Estrutura dobravel com case de transporte incluso facilita guardar na mochila ou mala sem ocupar espaco
```

### Palavras de Poder para Bullets

| Categoria | Exemplos |
|-----------|----------|
| Durabilidade | `resistente`, `duravel`, `robusto`, `construcao solida` |
| Facilidade | `facil de usar`, `intuitivo`, `pratico`, `rapido` |
| Conforto | `ergonomico`, `macio`, `confortavel`, `leve` |
| Tecnologia | `avancado`, `inteligente`, `automatico`, `preciso` |
| Resultado | `eficiente`, `poderoso`, `profissional`, `premium` |

## 3. DESCRICAO (Max 2,000 chars)

### Estrutura de Descricao

```
[Paragrafo 1 - Introducao]
[Produto] oferece [beneficio principal] para [publico-alvo].

[Paragrafo 2 - Diferencial]
Com [caracteristica unica], voce consegue [resultado desejado].

[Paragrafo 3 - Especificacoes]
Especificacoes: [lista de specs relevantes]

[Paragrafo 4 - Uso/Aplicacao]
Ideal para [situacoes de uso]. [Beneficio adicional].

[Paragrafo 5 - Conteudo/Garantia]
Inclui: [lista do pacote]. Garantia do fabricante de [periodo].
```

### Formatacao

- **SEM HTML** - Texto puro apenas
- Use A+ Content para formatacao visual
- Paragrafos curtos (2-3 frases)
- Informacoes objetivas e factuais

## 4. BACKEND KEYWORDS (250 bytes)

### O Que Sao?

Keywords ocultas que so a Amazon ve. Usadas para SEO sem poluir o titulo.

### Regras de Backend

| Regra | Correto | Errado |
|-------|---------|--------|
| Separador | Espaco | Virgula |
| Repeticao | NAO repetir | Mesma keyword 3x |
| Do titulo | NAO incluir | Duplicar titulo |
| Limite | 250 bytes | Exceder limite |

### Estrategia de Backend Keywords

```
1. Sinonimos nao usados no titulo
2. Erros comuns de digitacao
3. Variacoes regionais (brasileiro vs portugues)
4. Termos de busca alternativos
5. Categorias relacionadas
```

### Exemplo de Backend

**Produto**: Fone Bluetooth JBL

**Titulo usa**: `fone bluetooth jbl preto dobravel`

**Backend Keywords** (NAO repetir do titulo):
```
headphone wireless auricular sem fio headset gamer musica podcast audio som qualidade graves basico trabalho home office escritorio estudo portatil viagem aviao presente
```

## 5. A+ CONTENT (Brand Registry)

### Requisitos

- Conta Brand Registry aprovada
- Marca registrada no INPI
- Template Amazon aprovado

### Modulos Disponiveis

| Modulo | Uso Ideal |
|--------|-----------|
| Comparison Table | Comparar versoes do produto |
| Image with Text | Destacar beneficios visuais |
| Product Description | Storytelling da marca |
| Feature Focus | 1 feature em destaque |
| Image Carousel | Galeria de lifestyle |

### Estrutura Recomendada A+

```
[Modulo 1] Banner da marca + headline
[Modulo 2] 3-4 features com icones
[Modulo 3] Comparison table (se houver variantes)
[Modulo 4] Lifestyle images (produto em uso)
[Modulo 5] Specs tecnicas + garantia
```

### Impacto do A+ Content

- Aumento de conversao: **5-10%**
- Reducao de devolucoes: **15-20%**
- Melhora em SEO interno: Moderado

## 6. IMAGENS (7-9 sempre)

### Hierarquia de Imagens

| Posicao | Tipo | Requisito |
|---------|------|-----------|
| 1 (Principal) | Produto fundo branco | RGB 255,255,255, produto 85% |
| 2-3 | Angulos adicionais | Alta resolucao |
| 4-5 | Features/detalhes | Close-ups relevantes |
| 6-7 | Lifestyle/uso | Contexto real |
| 8 | Infografico | Dimensoes/specs |
| 9 | Embalagem | Conteudo do pacote |

### Regras da Imagem Principal

- Fundo **branco puro** (RGB 255,255,255)
- Produto ocupa **85% da area**
- **Sem texto ou graficos**
- **Sem acessorios** nao incluidos
- Produto **completo e em foco**

## 7. ERROS QUE CAUSAM REJEICAO

| Erro | Consequencia | Como Evitar |
|------|--------------|-------------|
| ALL CAPS no titulo | Rejeicao | Title case apenas |
| Preco no titulo | Rejeicao | Nunca mencionar |
| "Amazon" no titulo | Rejeicao | Nunca usar |
| Simbolos especiais | Rejeicao | Apenas letras/numeros |
| Fundo nao-branco img 1 | Rejeicao | RGB 255,255,255 |
| Claims nao comprovados | Suspensao | Apenas fatos |
| Backend > 250 bytes | Ignorado | Respeitar limite |

## 8. SEO AMAZON (A9 Algorithm)

### Fatores de Ranqueamento

| Fator | Peso | Como Otimizar |
|-------|------|---------------|
| Taxa conversao | 25-30% | Imagens + copy + preco |
| Historico vendas | 20-25% | Volume consistente |
| Preco competitivo | 10-15% | Monitorar concorrencia |
| Disponibilidade | 10-15% | Estoque sempre |
| Reviews/ratings | 15-20% | Qualidade + volume |
| Completude listing | 5-10% | Preencher tudo |

### Checklist SEO Amazon

- [ ] Titulo seguindo Style Guide (150-200 chars)
- [ ] 5 bullet points completos (150-250 chars cada)
- [ ] Descricao ate 2,000 chars
- [ ] Backend keywords otimizados (250 bytes)
- [ ] 7-9 imagens de alta qualidade
- [ ] Imagem principal fundo branco
- [ ] A+ Content (se Brand Registry)
- [ ] Video de demonstracao (opcional mas recomendado)
- [ ] Categoria correta selecionada
- [ ] Todos atributos preenchidos

## 9. FBA vs FBM

### Fulfillment by Amazon (FBA)

| Vantagem | Impacto |
|----------|---------|
| Selo Prime | Maior conversao |
| Frete rapido | Melhor experiencia |
| Ranqueamento | Favorecido pelo A9 |
| SAC | Amazon cuida |

### Fulfillment by Merchant (FBM)

| Vantagem | Quando Usar |
|----------|-------------|
| Controle total | Produtos frageis/grandes |
| Margem maior | Alto ticket medio |
| Flexibilidade | Personalizacao |
