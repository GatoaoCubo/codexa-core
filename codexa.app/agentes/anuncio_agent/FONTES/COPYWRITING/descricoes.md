# Descricoes - Frameworks e Estruturas

**Version**: 1.0.0 | **Focus**: StoryBrand + Gatilhos Mentais + Conversao

---

## Overview

A descricao converte visitantes em compradores. E o espaco para:
- **Storytelling**: Conexao emocional
- **Especificacoes**: Informacao tecnica
- **Persuasao**: Gatilhos mentais

---

## 1. FRAMEWORK STORYBRAND (7 Elementos)

### Os 7 Passos

```
1. HEROI → Cliente tem um problema
2. PROBLEMA → Interno, externo e filosofico
3. GUIA → Voce/produto como mentor
4. PLANO → Passos claros de solucao
5. CALL TO ACTION → Convite direto
6. SUCESSO → Visao do futuro positivo
7. FRACASSO → Consequencia de nao agir
```

### Aplicacao em Descricao

```markdown
[1. HEROI - Abrir com o cliente]
Voce que busca [objetivo/necessidade], sabe como e dificil [problema comum].

[2. PROBLEMA - 3 dimensoes]
Externo: [problema pratico/funcional]
Interno: [frustração/sentimento]
Filosofico: [por que isso importa]

[3. GUIA - Apresentar solucao]
Apresentamos [produto] - desenvolvido especialmente para [resolver problema].

[4. PLANO - 3-5 passos]
Como funciona:
1. [Passo 1 - simples]
2. [Passo 2 - facil]
3. [Passo 3 - resultado]

[5. CTA - Chamada clara]
Adicione ao carrinho agora e [beneficio imediato].

[6. SUCESSO - Pintar futuro positivo]
Imagine [cenario ideal apos uso do produto].

[7. FRACASSO - Consequencia sutil]
Nao deixe que [problema] continue atrapalhando [objetivo].
```

## 2. ESTRUTURA PADRAO (3,300+ chars)

### Bloco 1: Abertura (300-400 chars)

```markdown
[HEADLINE IMPACTANTE]

Voce ja [situacao problema]? Sabe como e [consequencia negativa]?

[Produto] foi criado para [resolver problema especifico].
Com [diferencial principal], voce finalmente consegue [resultado desejado].
```

### Bloco 2: Beneficios (600-800 chars)

```markdown
[POR QUE ESCOLHER ESTE PRODUTO?]

1. [BENEFICIO 1]: [explicacao com resultado tangivel]
   Exemplo: "Bateria de 40h: ouca musica o dia todo sem recarregar"

2. [BENEFICIO 2]: [explicacao com prova social]
   Exemplo: "Design premiado: reconhecido por especialistas da area"

3. [BENEFICIO 3]: [explicacao com diferencial tecnico]
   Exemplo: "Cancelamento de ruido ativo: foque no que importa"

4. [BENEFICIO 4]: [explicacao com conveniencia]
   Exemplo: "Dobravel e compacto: cabe em qualquer bolsa"

5. [BENEFICIO 5]: [explicacao com durabilidade]
   Exemplo: "Construcao premium: feito para durar anos"
```

### Bloco 3: Especificacoes (500-700 chars)

```markdown
[ESPECIFICACOES TECNICAS]

- Material: [material principal]
- Dimensoes: [largura] x [altura] x [profundidade]
- Peso: [peso em gramas/kg]
- Capacidade: [capacidade/volume]
- Potencia: [watts/voltagem]
- Conectividade: [bluetooth/wifi/cabo]
- Bateria: [duracao/mAh]
- Certificacoes: [INMETRO/ANVISA/ANATEL]
- Compatibilidade: [sistemas/dispositivos]
- Garantia: [periodo do fabricante]
```

### Bloco 4: Conteudo da Embalagem (200-300 chars)

```markdown
[CONTEUDO DA EMBALAGEM]

- 1x [Produto principal]
- 1x [Acessorio 1]
- 1x [Acessorio 2]
- 1x Manual de instrucoes
- 1x Certificado de garantia
```

### Bloco 5: Diferenciais (400-500 chars)

```markdown
[DIFERENCIAIS EXCLUSIVOS]

[Diferencial 1]: [explicacao do que torna unico]

[Diferencial 2]: [tecnologia ou inovacao]

[Diferencial 3]: [vantagem competitiva]

Por que somos diferentes: [resumo da proposta de valor]
```

### Bloco 6: Garantia e Confianca (300-400 chars)

```markdown
[GARANTIA E SEGURANCA]

Garantia: [periodo] direto com [fabricante/vendedor].

Compra Protegida: [mencionar protecao do marketplace].

Vendedor verificado com [X] avaliacoes positivas e [X] vendas realizadas.

Qualquer duvida, estamos a disposicao para ajudar!
```

### Bloco 7: CTA Final (200-300 chars)

```markdown
[APROVEITE AGORA!]

Nao perca a oportunidade de [transformacao/beneficio].

Adicione ao carrinho e receba [vantagem da compra].

Estoque limitado - garanta ja o seu!
```

## 3. GATILHOS MENTAIS

### Os 6 Gatilhos Principais

| Gatilho | Definicao | Aplicacao |
|---------|-----------|-----------|
| **Escassez** | Pouco disponivel | "Ultimas X unidades" |
| **Urgencia** | Tempo limitado | "Oferta valida ate..." |
| **Prova Social** | Outros confiam | "Mais de X vendidos" |
| **Autoridade** | Especialista | "Lider na categoria" |
| **Reciprocidade** | Dar antes | "Brinde exclusivo" |
| **Compromisso** | Pequenos passos | "Experimente 7 dias" |

### Exemplos de Uso

**Escassez**:
```
Ultimas 15 unidades em estoque!
Producao limitada - nao tera reposicao.
Edicao especial exclusiva.
```

**Urgencia**:
```
Oferta valida apenas hoje!
Preco promocional por tempo limitado.
Frete gratis so esta semana.
```

**Prova Social**:
```
Mais de 10.000 clientes satisfeitos.
Avaliacao 4.9/5 com base em 500+ reviews.
Produto mais vendido na categoria.
```

**Autoridade**:
```
Marca lider no segmento desde [ano].
Recomendado por [especialista/instituicao].
Tecnologia premiada internacionalmente.
```

**Reciprocidade**:
```
Brinde exclusivo: [item] incluso na compra.
E-book gratuito com dicas de uso.
Suporte prioritario para compradores.
```

**Compromisso**:
```
Satisfacao garantida ou dinheiro de volta.
Experimente 30 dias sem risco.
Primeiro mes com desconto especial.
```

## 4. FORMULAS DE COPY

### Formula PAS (Problem-Agitate-Solution)

```
[PROBLEMA]
Voce sofre com [problema comum]?

[AGITACAO]
Isso causa [consequencia 1], [consequencia 2] e [consequencia 3].
Quantas vezes voce ja [situacao frustrante]?

[SOLUCAO]
[Produto] resolve isso de uma vez por todas.
Com [mecanismo], voce consegue [resultado] sem [dificuldade anterior].
```

### Formula AIDA (Attention-Interest-Desire-Action)

```
[ATENCAO]
[Fato surpreendente ou pergunta impactante]

[INTERESSE]
[Beneficios e diferenciais que despertam curiosidade]

[DESEJO]
[Pintar cenario ideal - como a vida melhora com o produto]

[ACAO]
[CTA claro e direto]
```

### Formula FAB (Features-Advantages-Benefits)

```
[FEATURE]
Bateria de 5000mAh

[ADVANTAGE]
Dura 3x mais que baterias convencionais

[BENEFIT]
Voce usa o dia todo sem se preocupar com tomada
```

## 5. ADAPTACAO POR MARKETPLACE

### Mercado Livre (3,000-5,000 chars)

- HTML basico permitido (`<b>`, `<br>`, `<p>`)
- Descricao longa e detalhada
- Especificacoes tecnicas completas
- FAQ embutido opcional

### Shopee (1,500-2,500 chars)

- Texto puro apenas (sem HTML)
- Paragrafos curtos (mobile)
- Foco em beneficios visuais
- Emojis com moderacao

### Amazon (1,500-2,000 chars)

- Texto puro (usar A+ para visual)
- Tom profissional e factual
- Sem claims nao comprovados
- Especificacoes precisas

### Magalu (2,500-3,500 chars)

- Texto puro
- Ficha tecnica detalhada
- Certificacoes destacadas
- Garantia clara

## 6. LINGUAGEM PERSUASIVA

### Palavras que VENDEM

| Categoria | Palavras |
|-----------|----------|
| Beneficio | revolucionario, transformador, definitivo |
| Facilidade | simples, rapido, pratico, intuitivo |
| Exclusividade | unico, exclusivo, diferenciado, especial |
| Seguranca | garantido, protegido, certificado, seguro |
| Economia | economico, custo-beneficio, investimento |
| Qualidade | premium, profissional, superior, duravel |

### Palavras a EVITAR

| Palavra | Problema | Alternativa |
|---------|----------|-------------|
| Barato | Desvaloriza | Acessivel, economico |
| O melhor | Claim proibido | Um dos mais reconhecidos |
| Perfeito | Exagero | Otimo, excelente |
| 100% garantido | Risco legal | Satisfacao garantida |
| Cura/trata | Proibido ANVISA | Auxilia, contribui |

### Tom de Voz

| Situacao | Tom |
|----------|-----|
| Produtos premium | Sofisticado, exclusivo |
| Produtos populares | Acessivel, amigavel |
| Produtos tecnicos | Informativo, preciso |
| Produtos de beleza | Aspiracional, confiante |
| Produtos infantis | Seguro, divertido |

## 7. CHECKLIST DE DESCRICAO

### Obrigatorios

- [ ] Minimo 3,300 caracteres (ML padrao)
- [ ] StoryBrand 7 elementos presentes
- [ ] Pelo menos 3 gatilhos mentais
- [ ] Especificacoes tecnicas completas
- [ ] Conteudo da embalagem listado
- [ ] Garantia informada
- [ ] CTA claro no final

### Validacao de Compliance

- [ ] Zero claims medicos/terapeuticos
- [ ] Zero superlativos absolutos sem prova
- [ ] Zero links externos
- [ ] Zero precos inline
- [ ] Zero emojis (exceto Shopee)
- [ ] Zero HTML proibido

### Qualidade de Copy

- [ ] Foco no cliente (voce > nos)
- [ ] Beneficios > features
- [ ] Paragrafos curtos
- [ ] Escaneabilidade
- [ ] Linguagem clara

```
ESPECIFICACOES:
- [spec 1]
- [spec 2]
- [spec 3]

INCLUI:
- [item 1]

```

```
DETALHES:

- [diferencial 1]
```

```
- [lista de itens]
```
