# Titulos - Otimizacao e Formulas

**Version**: 1.0.0 | **Focus**: SEO + CTR + Zero Conectores

---

## Overview

O titulo e o elemento mais importante do anuncio. Determina:
- **SEO**: Ranqueamento nas buscas
- **CTR**: Taxa de clique
- **Relevancia**: Match com intencao de busca

---

## 1. REGRA DE OURO: ZERO CONECTORES

### O Que Sao Conectores?

Palavras que ligam termos mas nao agregam valor de busca:

| Conectores PROIBIDOS | Substituicao |
|----------------------|--------------|
| de | [espaco] |
| para | [espaco] |
| com | [espaco] |
| e | [espaco] |
| em | [espaco] |
| do/da | [espaco] |
| no/na | [espaco] |
| ao/a | [espaco] |

### Exemplos de Transformacao

| ERRADO (com conectores) | CORRETO (sem conectores) |
|-------------------------|--------------------------|
| Camiseta de Algodao para Academia | Camiseta Algodao Academia Masculina |
| Fone com Bluetooth e Microfone | Fone Bluetooth Microfone Dobravel |
| Panela de Pressao em Aluminio | Panela Pressao Aluminio 6L Eletrica |
| Kit de Ferramentas para Casa | Kit Ferramentas Casa 100 Pecas Profissional |

### Por Que Zero Conectores?

1. **Mais keywords** no limite de caracteres
2. **Melhor match** com buscas do usuario
3. **SEO otimizado** para algoritmo
4. **Densidade maior** de termos relevantes

## 2. FORMULAS POR MARKETPLACE

### Mercado Livre (60 chars)
```
[TIPO] + [ATRIBUTO 1] + [ATRIBUTO 2] + [DIFERENCIAL] + [SPEC]
```

**Exemplos**:
```
Whey Protein Isolado 1kg Chocolate Growth Importado
Fone Bluetooth JBL Tune 510 Original Preto Dobravel
Camiseta Dry Fit Masculina Academia UV50 Kit 3 Pecas
```

### Shopee (120 chars)
```
[PRODUTO] + [ATRIBUTOS] + [DIFERENCIAIS] + [VARIACOES] + [EXTRAS]
```

**Exemplos**:
```
Blusa Cropped Feminina Canelada Manga Curta Basica Casual Cores Variadas Verao

```
[MARCA] + [LINHA] + [CARACTERISTICA] + [MATERIAL] + [COR] + [TAMANHO] + [QTD]
```

**Exemplos**:
```

Tramontina Jogo de Panelas Turim Aluminio Antiaderente 5 Pecas Vermelho

## 3. DENSIDADE DE KEYWORDS

### Meta: 8-10 Keywords por Titulo

**Contagem de Exemplo**:
```
Whey(1) Protein(2) Isolado(3) 1kg(4) Chocolate(5) Growth(6) Importado(7) Lacrado(8)
= 8 keywords em 58 caracteres
```

### Priorizacao de Keywords

| Prioridade | Tipo | Exemplo |
|------------|------|---------|
| 1 | Head term (produto) | "Whey Protein" |
| 2 | Especificacao principal | "Isolado" |
| 3 | Quantidade/tamanho | "1kg" |
| 4 | Variacao (cor/sabor) | "Chocolate" |
| 5 | Marca | "Growth" |
| 6 | Diferencial | "Importado" |
| 7 | Atributo extra | "Lacrado" |
| 8+ | Long-tail | "Academia", "Treino" |

## 6. ERROS COMUNS

| Erro | Problema | Correcao |
|------|----------|----------|
| Titulo curto | Perde SEO | Usar 58-60 chars (ML) |
| Conectores | Desperdiça espaco | Remover todos |
| Keyword stuffing | Parece spam | Max 2x mesma palavra |
| ALL CAPS | Rejeitado | Title case |
| Emojis | Proibido | Remover todos |
| Precos | Proibido | Nunca no titulo |
| Links | Proibido | Nunca no titulo |

## 7. CHECKLIST PRE-PUBLICACAO

### Validacao Obrigatoria

- [ ] Zero conectores ("de", "para", "com", "e")
- [ ] Dentro do limite de caracteres
- [ ] 8-10 keywords presentes
- [ ] Head term no inicio
- [ ] Marca incluida (quando relevante)
- [ ] Especificacao principal presente
- [ ] Diferencial destacado
- [ ] Sem emojis ou simbolos especiais
- [ ] Sem ALL CAPS
- [ ] Sem precos ou promocoes

### Teste de Leitura

Pergunte-se:
1. O cliente entende o que e o produto?
2. As keywords mais buscadas estao presentes?
3. O diferencial esta claro?
4. Funciona em tela de celular?

## 9. VARIACAO A/B

### Estrategia de 3 Titulos

Sempre gerar 3 variacoes para teste:

**Variacao A** (SEO-focused):
```
Keywords mais buscados primeiro
Fone Bluetooth JBL Tune 510 Original Preto Dobravel Leve
```

**Variacao B** (Beneficio-focused):
```
Diferencial destacado
JBL Tune 510 Fone Bluetooth 40h Bateria Dobravel Preto
```

**Variacao C** (Marca-focused):
```
Marca no inicio (para brand search)
JBL Tune 510BT Fone Bluetooth Original Preto Dobravel Microfone
```

### Metricas de Teste

| Metrica | Bom | Otimo |
|---------|-----|-------|
| CTR | > 2% | > 4% |
| Conversao | > 3% | > 5% |
| Posicao media | Top 20 | Top 5 |
