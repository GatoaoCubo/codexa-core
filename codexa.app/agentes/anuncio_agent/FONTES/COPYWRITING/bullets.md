# Bullets - Patterns e Gatilhos

**Version**: 1.0.0 | **Focus**: 10 Bullets x 250-299 chars + Mental Triggers

---

## Overview

Bullets sao os pontos-chave que destacam beneficios do produto. Objetivos:
- **Escaneabilidade**: Leitura rapida
- **Persuasao**: Gatilhos em cada ponto
- **Conversao**: Convencer em segundos

---

## 1. ESPECIFICACOES TECNICAS

### Mercado Livre / Shopee / Magalu
- **Quantidade**: 10 bullets
- **Caracteres**: 250-299 por bullet
- **Formato**: Texto corrido (sem lista)

### Amazon Brasil
- **Quantidade**: 5 bullets
- **Caracteres**: 150-250 por bullet (max 500)
- **Formato**: Inicio com MAIUSCULA destaque

## 2. ESTRUTURA DO BULLET PERFEITO

### Formula Universal
```
[GATILHO] + [BENEFICIO] + [PROVA/DETALHE] + [RESULTADO]
```

### Componentes

| Componente | Funcao | Exemplo |
|------------|--------|---------|
| GATILHO | Chamar atencao | "BATERIA EXTRA-LONGA:" |
| BENEFICIO | O que ganha | "40 horas de musica" |
| PROVA | Como/porque | "com tecnologia avancada" |
| RESULTADO | Transformacao | "para usar o dia todo sem recarregar" |

### Exemplo Completo
```
BATERIA EXTRA-LONGA: Aproveite ate 40 horas de reproducao continua com uma unica carga,
perfeito para viagens longas e uso diario intenso sem precisar de tomada por varios dias.
(278 caracteres)
```

## 4. GATILHOS MENTAIS POR BULLET

### Gatilho: Transformacao
```
Estrutura: [Antes negativo] → [Depois positivo]
Exemplo: "Deixe para tras fones desconfortaveis e experimente o conforto premium que
transforma suas horas de uso em momentos de prazer auditivo absoluto."
```

### Gatilho: Autoridade
```
Estrutura: [Credencial] + [Beneficio]
Exemplo: "Tecnologia desenvolvida pelos mesmos engenheiros de marcas premiadas, garantindo
qualidade de audio profissional acessivel para uso domestico."
```

### Gatilho: Prova Social
```
Estrutura: [Numero/estatistica] + [Validacao]
Exemplo: "Escolha de mais de 50.000 clientes satisfeitos que avaliaram com 5 estrelas,
comprovando a excelencia em qualidade e atendimento."
```

### Gatilho: Escassez
```
Estrutura: [Limitacao] + [Consequencia]
Exemplo: "Producao limitada com poucas unidades restantes em estoque, garantindo
exclusividade para quem adquirir agora."
```

### Gatilho: Urgencia
```
Estrutura: [Tempo] + [Acao]
Exemplo: "Oferta especial disponivel por tempo limitado, aproveite condicoes exclusivas
que nao serao repetidas."
```

### Gatilho: Reciprocidade
```
Estrutura: [Brinde/bonus] + [Valor]
Exemplo: "Inclui case de transporte premium no valor de R$ 89 totalmente gratis,
presente exclusivo para compradores desta oferta."
```

### Gatilho: Seguranca
```
Estrutura: [Garantia] + [Tranquilidade]
Exemplo: "Satisfacao garantida com 30 dias para testar, se nao atender suas expectativas
devolvemos seu dinheiro sem questionamentos."
```

## 5. FORMULAS DE BULLETS

### Formula BENEFIT-PROOF-RESULT
```
[O que voce ganha] + [Como/porque funciona] + [O resultado na sua vida]
```

### Formula PROBLEM-SOLUTION-OUTCOME
```
[Problema comum] + [Como o produto resolve] + [Sua nova realidade]
```

### Formula FEATURE-ADVANTAGE-BENEFIT
```
[Caracteristica tecnica] + [Vantagem pratica] + [Beneficio emocional]
```

### Exemplos Praticos

**BENEFIT-PROOF-RESULT**:
```
Duracao de bateria excepcional de 40 horas (BENEFIT) gracas a tecnologia de otimizacao
energetica avancada (PROOF), permitindo semanas de uso sem se preocupar com recarga (RESULT).
```

**PROBLEM-SOLUTION-OUTCOME**:
```
Cansado de fones que machucam apos 1 hora? (PROBLEM) Almofadas ergonomicas em memory foam
distribuem pressao uniformemente (SOLUTION), garantindo conforto mesmo em sessoes de 8 horas (OUTCOME).
```

**FEATURE-ADVANTAGE-BENEFIT**:
```
Drivers de 40mm de diametro (FEATURE) produzem resposta de frequencia ampla de 20Hz-20kHz
(ADVANTAGE), transportando voce para dentro da musica como se estivesse no estudio (BENEFIT).
```

## 7. ERROS COMUNS

| Erro | Problema | Correcao |
|------|----------|----------|
| Bullet curto demais | Perde impacto | Min 250 chars |
| Bullet longo demais | Perde leitura | Max 299 chars |
| Foco em features | Nao converte | Focar em beneficios |
| Repeticao | Entedia leitor | Variar gatilhos |
| Sem gatilho | Nao persuade | 1 gatilho por bullet |
| Generico | Nao diferencia | Especificar numeros |
| ALL CAPS excessivo | Parece spam | Apenas 1-2 palavras |

## 8. CHECKLIST DE BULLETS

### Validacao Individual (cada bullet)

- [ ] 250-299 caracteres
- [ ] Inicia com destaque/gatilho
- [ ] Contem beneficio claro
- [ ] Tem prova ou detalhe
- [ ] Termina com resultado/transformacao
- [ ] Linguagem persuasiva

### Validacao do Conjunto (10 bullets)

- [ ] Todos os 10 bullets presentes
- [ ] Gatilhos variados (nao repetir)
- [ ] Cobertura completa do produto
- [ ] Progressao logica
- [ ] Bullet 1 = mais forte
- [ ] Bullet 10 = CTA/urgencia
- [ ] Zero claims proibidos

## 9. TEMPLATES DE BULLETS

proporcionando [resultado/transformacao] para [publico/situacao de uso].

```
[SPEC TECNICA]: [Numero/dado] que garante [vantagem pratica],
superando [comparacao] e entregando [beneficio] em qualquer [cenario de uso].
```

### Template Emocional
```

finalmente voce consegue [desejo] sem [dor/problema anterior].
```

### Template Prova Social
```

comprovando [qualidade/beneficio] que transforma [area da vida].
```
