## TIPOS DE VOMITO

### 1. Vomito Pos-Refeicao Rapida

- Logo apos comer (minutos)
- Racao praticamente inteira
- Gato come muito rapido
- **Causa principal**: Ansiedade alimentar / competicao

### 2. Regurgitacao

- Comida NAO digerida
- Sem esforco abdominal
- Pode indicar problema esofagico
- **Causa**: mecanica, nao digestiva

### 3. Vomito Verdadeiro

- Esforco abdominal visivel
- Conteudo parcialmente digerido
- Bile amarela/verde
- **Causa**: gastrica ou sistemica

### 4. Bolas de Pelo

- Pelo visivel no vomito
- Normal 1-2x por mes
- Excessivo = problema de pelagem/grooming

---

## RED FLAGS (Veterinario URGENTE)

```
⚠️ ENCAMINHAR IMEDIATAMENTE SE:
- Sangue no vomito (vermelho ou marrom escuro)
- Vomito + letargia
- Vomito + diarreia
- Vomito + anorexia (>24h sem comer)
- Vomito + perda de peso
- Vomito frequente (>3x semana)
- Suspeita de intoxicacao
- Abdomen distendido/doloroso
```

## CAUSAS COMUNS

### Comportamentais

| Causa | Sinais | Solucao |
|-------|--------|---------|
| Comer rapido | Vomita logo apos | Comedouro lento |
| Competicao | Multi-gatos | Separar alimentacao |
| Ansiedade | Gato tenso | Ambiente calmo |
| Comida fria | Racao umida gelada | Esquentar levemente |

### Medicas

| Causa | Sinais | Acao |
|-------|--------|------|
| Bola de pelo | Pelo no vomito | Pasta malte + escovacao |
| Intolerancia | Sempre com X comida | Trocar proteina |
| Gastrite | Bile amarela/manha | Vet + alimentacao fracionada |
| Parasitas | Gato jovem/rua | Vermifugo |
| Pancreatite | Dor abdominal | Emergencia vet |

## SOLUCOES POR PRIORIDADE

### 1. Descartar Emergencia

```
SE qualquer red flag → Veterinario imediato
NAO tentar resolver em casa
```

### 2. Ajustar Alimentacao

```
✓ Comedouro lento (obstaculos)
✓ Porcoes menores, mais vezes ao dia
✓ Altura adequada (10-15cm do chao)
✓ Local tranquilo, longe de outros pets
✓ Racao umida em temperatura ambiente
```

### 3. Reduzir Competicao

```
✓ Alimentar gatos separadamente
✓ Cada gato com seu prato
✓ Distancia minima 1-2 metros
✓ Supervisionar refeicoes
```

### 4. Manejo de Bolas de Pelo

```
✓ Escovar 2-3x por semana
✓ Pasta de malte 2-3x por semana
✓ Racao para bolas de pelo
✓ Aumentar fibra na dieta
```

## COMEDOURO IDEAL

```
Caracteristicas:
- Raso e largo (bigodes nao tocam)
- Elevado 10-15cm (digestao)
- Material: ceramica ou aco inox
- Lavar diariamente

Para comedores rapidos:
- Com obstaculos internos
- Tipo puzzle/labirinto
- Porcoes menores
```

---

## O QUE NAO FAZER

- ❌ Deixar em jejum prolongado (hepatite lipidica)
- ❌ Dar leite (maioria eh intolerante)
- ❌ Trocar racao bruscamente
- ❌ Ignorar vomito frequente
- ❌ Automeditar

### Tom: Acolhedor + Cauteloso

```
"Entendo sua preocupacao com o vomito! Primeiro, vamos
verificar: tem sangue? Seu gato esta apatico ou sem apetite?
Se sim, veterinario urgente.

Se ele vomita logo apos comer mas parece bem depois,
provavelmente esta comendo rapido demais - super comum!

Solucoes praticas:
1. Comedouro lento (com obstaculos)
2. Dividir racao em 3-4 refeicoes menores
3. Elevar o prato 10-15cm do chao

Se persistir apos 1 semana de ajustes, melhor consultar o vet!"
```
