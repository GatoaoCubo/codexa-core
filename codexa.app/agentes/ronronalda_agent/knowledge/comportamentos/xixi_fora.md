# XIXI FORA DA CAIXA | Eliminacao Inadequada

> Base de conhecimento para Ronronalda
> Ultima atualizacao: 2025-11-28

---

## TIPOS DE PROBLEMA

### 1. Marcacao Territorial (Spraying)

- Urina em superficies VERTICAIS
- Pequenas quantidades
- Gato fica de pe, cauda tremendo
- Mais comum em machos nao castrados
- Gatilho: estresse, novo gato, mudancas

### 2. Eliminacao Inadequada

- Urina em superficies HORIZONTAIS
- Quantidades normais
- Postura agachada normal
- Problema com a CAIXA ou localizacao

---

## CAUSAS MAIS COMUNS

### Medicas (Descartar PRIMEIRO!)

- Infeccao urinaria (cistite)
- Cristais/pedras na bexiga
- Diabetes
- Doenca renal
- Artrite (dor ao entrar na caixa)

**SEMPRE encaminhar ao vet se:**
- Urina com sangue
- Miados ao urinar
- Lambendo genitais excessivamente
- Mudanca subita de comportamento

### Comportamentais

| Causa | Solucao |
|-------|---------|
| Caixa suja | Limpar 1-2x por dia |
| Areia errada | Testar areia sem perfume |
| Caixa pequena | Caixa = 1.5x tamanho do gato |
| Local ruim | Mover para local tranquilo |
| Poucos recursos | Regra N+1 (2 gatos = 3 caixas) |
| Estresse territorial | Avaliar conflito entre gatos |
| Caixa fechada | Alguns gatos nao gostam |

---

## REGRA DE OURO

```
N gatos = N + 1 caixas

Exemplos:
- 1 gato = 2 caixas
- 2 gatos = 3 caixas
- 3 gatos = 4 caixas

DISTRIBUIDAS pela casa (nao lado a lado!)
```

## CHECKLIST DE INVESTIGACAO

```
□ Quando comecou? (evento estressor?)
□ Sempre no mesmo local?
□ Vertical (spraying) ou horizontal?
□ Gato castrado?
□ Quantas caixas existem?
□ Onde estao localizadas?
□ Tipo de areia usada?
□ Frequencia de limpeza?
□ Outros gatos na casa?
□ Mudancas recentes? (moveis, pessoas, rotina)
```

---

## SOLUCOES POR PRIORIDADE

### 1. Descartar Medico

```
Encaminhar ao veterinario SE:
- Sangue na urina
- Dor ao urinar
- Mudanca subita
- Gato idoso
- Aumento de frequencia
```

### 2. Otimizar Caixas

```
✓ Quantidade: N+1
✓ Tamanho: 1.5x comprimento do gato
✓ Areia: sem perfume, 5-7cm de profundidade
✓ Limpeza: retirar solidos 1-2x/dia
✓ Troca total: semanal
✓ Local: tranquilo, escape facil, longe da comida
```

### 3. Limpar Locais Marcados

```
✓ Limpar com enzimatico (nao amonia!)
✓ Cobrir temporariamente com plastico
✓ Colocar comedouro no local (gatos nao urinam onde comem)
✓ Usar feromonio sintetico (Feliway)
```

### 4. Reduzir Estresse

```
✓ Mais recursos (caixas, agua, arranhadores)
✓ Espacos verticais (prateleiras, torres)
✓ Esconderijos seguros
✓ Rotina previsivel
```

---

## O QUE NAO FAZER

- ❌ Esfregar focinho na urina
- ❌ Punir (aumenta estresse = piora problema)
- ❌ Limpar com amonia (cheiro similar a urina)
- ❌ Ignorar (pode ser problema medico)
- ❌ Trancar em comodo

## CASOS ESPECIAIS

### Gato Idoso

- Artrite pode dificultar entrada na caixa
- Usar caixa com entrada baixa
- Colocar mais caixas (mobilidade reduzida)

### Multi-gatos

- Conflito silencioso pode causar marcacao
- Garantir recursos suficientes
- Observar dinamica social

### Mudanca Recente

- Nova casa, bebe, pet = estresse
- Dar tempo de adaptacao
- Manter rotina o mais estavel possivel
