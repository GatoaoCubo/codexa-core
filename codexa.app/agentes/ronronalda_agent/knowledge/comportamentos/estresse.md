## ENTENDENDO O ESTRESSE FELINO

Gatos sao **criaturas de habito**. Qualquer mudanca pode gerar estresse:
- Novo ambiente
- Novo membro (humano, pet, bebe)
- Mudanca de rotina
- Conflito com outros animais
- Falta de recursos
- Ambiente pobre em estimulos

---

## SINAIS DE ESTRESSE

### Comportamentais

| Sinal | O que significa |
|-------|-----------------|
| Esconder-se excessivamente | Busca seguranca |
| Agressividade subita | Defesa por medo |
| Vocalizacao excessiva | Pedido de ajuda |
| Xixi fora da caixa | Marcacao de estresse |
| Comer menos/mais | Alteracao de apetite |
| Nao brincar | Apatia |

### Fisicos

| Sinal | O que significa |
|-------|-----------------|
| Grooming excessivo | Auto-calmante (pode causar falhas) |
| Pupilas dilatadas | Estado de alerta |
| Orelhas para tras | Medo/tensao |
| Cauda baixa | Inseguranca |
| Pelos ericados | Defensivo |
| Respiracao rapida | Ansiedade aguda |

## CAUSAS COMUNS

### Mudancas Ambientais

```
- Mudanca de casa
- Reforma/obras
- Moveis novos
- Cheiros diferentes
- Visitantes frequentes
```

### Recursos Insuficientes

```
- Poucas caixas de areia
- Poucos pontos de agua
- Falta de espacos altos
- Falta de esconderijos
- Territorio insuficiente
```

### Conflitos Sociais

```
- Novo gato (apresentacao ruim)
- Gatos incompativeis
- Cachorro perseguindo
- Crianca incomodando
- Gato de rua na janela
```

### Rotina Imprevisivel

```
- Horarios variaveis
- Ausencias longas
- Muita agitacao
- Barulhos altos
```

---

## AS 5 NECESSIDADES ESSENCIAIS

1. **Territorio seguro** - Espacos onde se sente protegido
2. **Recursos distribuidos** - Comida, agua, caixa em locais separados
3. **Enriquecimento** - Brinquedos, arranhadores, estimulos
4. **Interacao positiva** - Brincadeiras, carinho (quando quer)
5. **Previsibilidade** - Rotina estavel

---

## SOLUCOES POR PRIORIDADE

### 1. Criar Zonas de Refugio

```
✓ Prateleiras altas (gatos amam altura)
✓ Tocas/casinhas em locais tranquilos
✓ Caixas de papelao (simples e eficaz)
✓ Cobertores em locais escondidos
✓ Comodo "seguro" so do gato
```

### 2. Aumentar Recursos

```
✓ Regra N+1 para caixas de areia
✓ Multiplos pontos de agua
✓ Alimentacao em local calmo
✓ Arranhadores em varios comodos
```

### 3. Enriquecimento Ambiental

```
✓ Brinquedos rotativos (trocar a cada semana)
✓ Sessoes de brincadeira 2x/dia, 15min
✓ Puzzle feeders (estimulo mental)
✓ Janela com vista segura
✓ Videos para gatos (com supervisao)
```

### 4. Feromonios Sinteticos

```
✓ Feliway Classic (difusor)
✓ Feliway Friends (multi-gatos)
✓ Spray em transportadora/cama
✓ Uso continuo por 30 dias minimo
```

### 5. Rotina Previsivel

```
✓ Horarios fixos de alimentacao
✓ Horarios fixos de brincadeira
✓ Transicoes graduais
✓ Calma antes de sair/chegar
```

## ADAPTACAO A MUDANCAS

### Nova Casa

```
Semana 1: Confinar em 1 comodo com todos recursos
Semana 2: Expandir gradualmente
Semana 3+: Acesso total, manter comodo seguro

Dicas:
- Levar itens com cheiro familiar
- Manter rotina anterior
- Nao forcar exploracao
```

### Novo Pet

```
Fase 1: Separacao total (cheiro sob porta)
Fase 2: Troca de ambientes (sem contato visual)
Fase 3: Contato visual com barreira
Fase 4: Contato supervisionado curto
Fase 5: Aumento gradual de tempo junto

Tempo: 2-4 semanas (ou mais!)
```

### Novo Bebe

```
- Preparar ANTES do nascimento
- Familiarizar com sons/cheiros
- Manter rotina do gato
- Criar espacos de refugio
- Nunca deixar sozinhos
```
