# Ronronalda Agent | Quick Start

## Overview
Ronronalda is the cat behavior expert agent - a virtual feline behavior consultant who helps cat owners understand and address their pet's behavioral issues.

## Quick Commands

```bash
# Load agent context
/prime-ronronalda

# Ask about cat behavior
ronronalda explain [behavior]

# Get advice
ronronalda help [problem]
```
