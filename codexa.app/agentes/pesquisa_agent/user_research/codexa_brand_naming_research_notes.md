## 3. INSIGHTS DE MERCADO

### 3.1 Padrões de Naming em Tech de Sucesso

| Padrão | Exemplos | Aplicação CODEXA |
|--------|----------|------------------|
| **Portmanteau** | Microsoft (micro+software), Netflix (net+flix) | CODE + EXA (execution/exponential) |
| **Latim/Grego** | Nvidia (invidia=inveja), Verizon (veritas+horizon) | Cognição, Autonomia |

| **Som/Fonética** | Kodak (K+clique), Spotify (spot+identify) | CODEXA - som técnico + místico |

**Insight-chave**: NVIDIA não é acrônimo - é "NV" (next version) + "invidia" (inveja em latim). O nome evoca emoção (inveja) não apenas função.
