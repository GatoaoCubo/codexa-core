# Composition Rules for Product Photography

**Version**: 1.0.0 | **Updated**: 2025-12-05 | **Domain**: Visual Composition Theory

---

## OVERVIEW

Regras de composicao sao principios visuais que guiam o posicionamento de elementos em uma imagem para criar impacto maximo. Este guia cobre as principais tecnicas aplicaveis a fotografia de produto.

---

## REGRAS FUNDAMENTAIS

### 1. Regra dos Tercos (Rule of Thirds)

**Conceito**: Divida a imagem em 9 partes iguais com 2 linhas horizontais e 2 verticais. Posicione elementos importantes nas interseccoes ou ao longo das linhas.

```
+---+---+---+
|   | * |   |     * = Pontos de interesse
+---+---+---+
| * |   | * |
+---+---+---+
|   | * |   |
+---+---+---+
```

**Aplicacao em Produto**:
- Produto na interseccao esquerda ou direita
- Feature importante alinhada com linha
- Espaco para copy em marketing

**Prompt Keywords**:
```
rule of thirds composition, product positioned at left third,
focal point at intersection, asymmetrical balance
```

**Quando Usar**:
- Lifestyle shots
- Marketing banners
- Social media posts
- Hero images com texto

---

### 2. Composicao Centralizada (Centered)

**Conceito**: Produto exatamente no centro do frame. Cria simetria e foco absoluto.

```
+---+---+---+
|   |   |   |
+---+---+---+
|   | X |   |     X = Produto centralizado
+---+---+---+
|   |   |   |
+---+---+---+
```

**Aplicacao em Produto**:
- Pack shots para marketplace
- Imagens principais de listagem
- Simetria enfatiza forma

**Prompt Keywords**:
```
centered composition, product in exact center of frame,
symmetrical framing, balanced placement
```

**Quando Usar**:
- Mercado Livre main image
- Amazon main image
- Produtos simetricos
- Clean minimalist look

---

### 3. Proporcao Aurea (Golden Ratio)

**Conceito**: Espiral matematica baseada em phi (1.618). Naturalmente agradavel ao olho humano.

```
+---------------+
|      +----+   |
|      |    |   |
|  +---+    |   |
|  |        |   |
|  +--------+   |
+---------------+
    [Espiral fibonacci]
```

**Aplicacao em Produto**:
- Posicionar produto no ponto focal da espiral
- Props seguem a curva
- Composicao organica

**Prompt Keywords**:
```
golden ratio composition, fibonacci spiral placement,
natural visual flow, phi proportion
```

**Quando Usar**:
- Editorial photography
- Premium branding
- Produtos naturais/organicos
- Arte e design

---

### 4. Linhas Guia (Leading Lines)

**Conceito**: Linhas na imagem direcionam o olhar para o produto.

```
      ↘
  ----→ [PRODUTO]
      ↗
```

**Aplicacao em Produto**:
- Bordas de mesa apontando para produto
- Padrao de props criando direcao
- Arquitetura do ambiente

**Prompt Keywords**:
```
leading lines directing attention to product,
visual pathway to subject, converging lines
```

**Quando Usar**:
- Lifestyle shots
- Contexto ambiental
- Narrativa visual
- Hierarchy emphasis

---

### 5. Espaco Negativo (Negative Space)

**Conceito**: Area vazia ao redor do produto. Menos e mais.

```
+-------------------+
|                   |
|     [produto]     |
|                   |
+-------------------+
```

**Aplicacao em Produto**:
- Destaca produto isolado
- Espaco para texto/logo
- Minimalismo premium

**Prompt Keywords**:
```
generous negative space, minimalist composition,
product isolated with breathing room, clean empty background
```

**Quando Usar**:
- Hero shots
- Anuncios com texto
- Premium positioning
- E-commerce listings

---

### 6. Simetria e Padrao

**Conceito**: Elementos espelhados ou repetidos criam ordem visual.

```
+---+---+---+
| A | B | A |     Simetria bilateral
+---+---+---+

+---+---+---+
| A | A | A |     Padrao repetido
+---+---+---+
```

**Aplicacao em Produto**:
- Produtos com design simetrico
- Colecoes/variantes lado a lado
- Flat lays organizados

**Prompt Keywords**:
```
symmetrical composition, balanced mirror arrangement,
geometric pattern, repeated elements
```

**Quando Usar**:
- Produtos simetricos (garrafas, embalagens)
- Collection shots
- Pattern-focused designs
- Satisfying visuals

---

### 7. Enquadramento (Framing)

**Conceito**: Usar elementos da cena para "emoldurar" o produto.

```
+---[       ]---+
|   [PRODUTO]   |
+---[       ]---+
```

**Aplicacao em Produto**:
- Arco de porta com produto
- Plantas emoldurando
- Props criando borda

**Prompt Keywords**:
```
framed composition, product surrounded by [elements],
natural frame created by environment
```

**Quando Usar**:
- Lifestyle contextual
- Storytelling
- Focus attention
- Depth creation

## HIERARQUIA VISUAL

### Primario, Secundario, Terciario

```
1. PRIMARIO   = Produto principal (maior, mais contrastante)
2. SECUNDARIO = Props de suporte (menores, complementares)
3. TERCIARIO  = Background/ambiente (sutil, nao compete)
```

**Implementacao**:
```
Prompt: "[main product] as primary subject, [supporting props]
as secondary elements, [background] providing subtle context"
```

### Contraste para Hierarquia

| Tipo | Efeito |
|------|--------|
| Tamanho | Maior = mais importante |
| Cor | Saturado = destaque |
| Foco | Sharp = atencao |
| Posicao | Centro/frontal = primario |
| Iluminacao | Mais iluminado = focal |

## BALANCO VISUAL

### Balanco Simetrico

```
Peso igual em ambos os lados.
Formal, estavel, tradicional.

[peso] === [peso]
```

**Prompt**: "symmetrical balance, equal visual weight on both sides"

### Balanco Assimetrico

```
Pesos diferentes equilibrados por posicao.
Dinamico, interessante, moderno.

[peso grande] ... [peso pequeno distante]
```

**Prompt**: "asymmetrical balance, large element balanced by distant smaller elements"

### Balanco Radial

```
Elementos irradiam do centro.
Energia, movimento, foco central.

      *
    * X *
      *
```

**Prompt**: "radial balance, elements emanating from central product"

## PROFUNDIDADE E CAMADAS

### Foreground, Midground, Background

```
FOREGROUND  → Elementos mais proximos (blur opcional)
MIDGROUND   → Produto em foco (sharp)
BACKGROUND  → Contexto distante (blur comum)

[fg] → [PRODUTO] → [bg]
```

**Prompt Keywords**:
```
three-layer depth composition, foreground props slightly
blurred, product in sharp midground focus, soft background bokeh
```

### Criando Profundidade

1. **Overlap**: Elementos se sobrepoe
2. **Size Difference**: Menores = mais distantes
3. **Focus Gradient**: Sharp to blur
4. **Atmospheric Perspective**: Menos contraste ao fundo
5. **Positioning**: Diagonal placement

## DIRECAO E MOVIMENTO

### Direcao do Olhar (Gaze Direction)

```
Se imagem tem pessoa, olhar direciona atencao.

Pessoa olhando → [PRODUTO]
```

**Prompt**: "model looking toward product, gaze directing attention"

### Movimento Implicito

```
Linhas diagonais sugerem dinamismo.
Horizontais = calma, estabilidade.
Verticais = forca, altura.
```

**Prompt Keywords**:
```
dynamic diagonal composition, sense of movement,
stable horizontal lines, powerful vertical elements
```

## RELACAO DE ASPECTO

### Aspect Ratios Comuns

| Ratio | Uso | Platforms |
|-------|-----|-----------|
| 1:1 | Quadrado | ML, Amazon, Instagram feed |
| 4:5 | Portrait | Instagram optimal |
| 16:9 | Landscape | Banners, YouTube |
| 9:16 | Vertical | Stories, TikTok, Reels |
| 3:4 | Portrait | Pinterest |
| 2:3 | Portrait | Traditional photo |

### Composicao por Ratio

**1:1 (Square)**:
```
- Centralized works well
- Rule of thirds adaptable
- Limited horizontal storytelling
```

**16:9 (Wide)**:
```
- Horizontal rule of thirds
- Space for text on sides
- Panoramic context
```

**9:16 (Tall)**:
```
- Vertical thirds
- Stack elements top to bottom
- Mobile-first thinking
```

## CORES NA COMPOSICAO

### Cores Dominantes vs Acentos

```
DOMINANTE (60-70%) → Fundo, areas grandes
SECUNDARIA (20-30%) → Produto, elementos medios
ACENTO (5-10%) → Detalhes que chamam atencao
```

### Contrastes de Cor

| Tipo | Efeito | Uso |
|------|--------|-----|
| Complementares | Alto contraste | Chamar atencao |
| Analogos | Harmonia | Sofisticacao |
| Monocromaticos | Elegancia | Premium |
| Triada | Vibrante | Energetico |

**Prompt Example**:
```
"product with [PRIMARY COLOR] as dominant, [SECONDARY] accents,
[BACKGROUND COLOR] providing contrast, cohesive color palette"
```

## COMPOSICAO POR CENA

### Scene 1: Hero White

```
Composicao: Centralizada
Espaco Negativo: Maximo
Hierarquia: Produto unico focal
Fundo: Branco puro
```

### Scene 2: Lifestyle

```
Composicao: Regra dos Tercos
Camadas: 3 niveis de profundidade
Hierarquia: Produto > Pessoa > Ambiente
Leading Lines: Do ambiente para produto
```

### Scene 3: Detail Macro

```
Composicao: Centralizada ou off-center
Foco: Seletivo na feature
Espaco Negativo: Blur circundante
Hierarquia: Detalhe e unico elemento
```

### Scene 4: Use Case

```
Composicao: Regra dos Tercos
Contexto: Ambiente de uso visivel
Balanco: Assimetrico (produto + pessoa)
Narrativa: Acao/uso em progresso
```

### Scene 5: Emotional Benefit

```
Composicao: Golden Ratio ou Tercos
Mood: Atmosfera > tecnica
Cor: Paleta emocional
Iluminacao: Dramatica/mood-setting
```

### Scene 6: Scale Reference

```
Composicao: Centralizada
Elementos: Produto + referencia
Hierarquia: Igual peso visual
Clareza: Comparacao obvia
```

### Scene 7: Environment Context

```
Composicao: Wide shot, tercos
Camadas: Multiplas
Balanco: Produto no ambiente
Aspecto: 16:9 ou panoramico
```

### Scene 8: Material Detail

```
Composicao: Tight crop
Foco: Textura em destaque
Iluminacao: Revela material
Espaco: Minimo em volta
```

### Scene 9: Commercial Hero

```
Composicao: Centralizada (como Scene 1)
Espaco: Para branding/logo
Fundo: Branco puro
Qualidade: Maxima (pack shot final)
```
