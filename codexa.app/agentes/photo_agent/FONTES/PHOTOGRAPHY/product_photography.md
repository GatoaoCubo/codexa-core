**Version**: 1.0.0 | **Updated**: 2025-12-05 | **Domain**: E-commerce Product Shots

## OVERVIEW

Este guia cobre tecnicas fundamentais de fotografia de produto para e-commerce, otimizadas para geracao de prompts AI.

---

## TIPOS DE SHOTS

### 1. Hero Shot (Pack Shot)

**Proposito**: Imagem principal do produto, primeira impressao

**Caracteristicas**:
- Fundo branco puro (#FFFFFF)
- Produto centralizado
- Foco total (f/8-f/11)
- Sem sombras ou reflexos
- Produto ocupa 85% do frame

**Prompt Elements**:
```
centered composition, pure white seamless backdrop, product occupies
85% of frame, no shadows, isolated product, sharp focus throughout,
50mm lens f/8, commercial e-commerce photography
```

**Use For**: Mercado Livre main, Amazon main, Shopee thumbnail

---

### 2. 45-Degree Angle Shot

**Proposito**: Mostrar dimensionalidade e profundidade

**Caracteristicas**:
- Angulo 45 graus horizontal
- Revela forma 3D
- Ligeira perspectiva
- Pode ter sombra suave

**Prompt Elements**:
```
45-degree angle view showing depth and dimension, three-quarter
perspective, subtle gradient background, soft shadow indicating
form, product depth visible
```

**Use For**: Carousel secundario, gallery images

---

### 3. Lifestyle Shot

**Proposito**: Produto em contexto de uso real

**Caracteristicas**:
- Ambiente natural (casa, escritorio, natureza)
- Modelo/pessoa opcional
- Profundidade de campo rasa
- Atmosfera emocional

**Prompt Elements**:
```
lifestyle photography, natural environment, shallow depth of field,
85mm lens bokeh, authentic moment, warm atmosphere, product in
natural use context
```

**Use For**: Social media, brand storytelling, Instagram

---

### 4. Detail/Macro Shot

**Proposito**: Destacar qualidade, textura, acabamento

**Caracteristicas**:
- Close-up extremo
- Textura visivel
- Foco seletivo
- Revela craftsmanship

**Prompt Elements**:
```
extreme macro close-up, 100mm macro lens f/2.8, texture detail
visible, ring light illumination, hyper-detailed surface,
premium quality materials
```

**Use For**: Quality demonstration, material showcase

---

### 5. Scale Reference Shot

**Proposito**: Demonstrar tamanho real do produto

**Caracteristicas**:
- Objeto de referencia conhecido (mao, moeda, regua)
- Proporcoes claras
- Fundo neutro
- Foco em comparacao

**Prompt Elements**:
```
product next to [reference object] for scale, size comparison
clearly visible, accurate proportions, informative composition,
neutral background
```

**Use For**: Produtos pequenos ou grandes, evitar devolucoes

---

### 6. Flat Lay (Top-Down)

**Proposito**: Visao overhead, popular em social media

**Caracteristicas**:
- Angulo 90 graus (direto de cima)
- Composicao geometrica
- Props complementares
- Estilo Instagram

**Prompt Elements**:
```
flat lay overhead view, 90-degree top-down perspective,
symmetrical arrangement, complementary props, even diffused
lighting, Instagram aesthetic
```

**Use For**: Pinterest, Instagram feed, collections

---

### 7. Group/Collection Shot

**Proposito**: Mostrar variantes ou kit completo

**Caracteristicas**:
- Multiplos itens organizados
- Hierarquia visual clara
- Cores/tamanhos visíveis
- Composicao balanceada

**Prompt Elements**:
```
product collection arranged aesthetically, multiple color variants
visible, organized grouping, balanced composition, clear visual
hierarchy
```

**Use For**: Kit products, color options, family shots

## ILUMINACAO

### Setups de Estudio

#### 1. Soft Box (Difusa)

```
Setup:
- 2-3 soft boxes posicionadas
- Luz principal frontal ou 45 graus
- Fill light oposta
- Iluminacao uniforme

Prompt: "soft box diffused lighting, even illumination,
no harsh shadows, professional studio setup"
```

**Best For**: Hero shots, product details, white background

#### 2. Three-Point Lighting

```
Setup:
- Key light (principal, mais forte)
- Fill light (suaviza sombras)
- Back/rim light (separacao do fundo)

Prompt: "three-point lighting setup, key light from left,
fill light from right, rim light creating edge definition"
```

**Best For**: Dimensional products, premium look

#### 3. Ring Light

```
Setup:
- Luz circular ao redor da camera
- Iluminacao frontal uniforme
- Ideal para macro

Prompt: "ring light illumination, even frontal lighting,
soft shadow ring, macro photography setup"
```

**Best For**: Macro/detail shots, jewelry, small items

#### 4. Natural Window Light

```
Setup:
- Janela como fonte principal
- Refletor para fill
- Difusor para suavizar

Prompt: "natural window light from [direction], soft diffused
daylight, authentic atmosphere, [time of day] quality"
```

**Best For**: Lifestyle shots, food, handcrafted items

### Direcao da Luz

| Direcao | Efeito | Uso |
|---------|--------|-----|
| Frontal | Flat, sem sombras | Pack shots |
| 45 graus | Dimensional, forma | General products |
| Lateral | Dramatico, textura | Artisanal items |
| Backlit | Silhouette, rim | Transparents, glass |
| Top-down | Flat lay, shadows | Overhead compositions |

### Temperatura de Cor

```
3000K (Warm)     → Aconchegante, cozy, inviting
4000K (Neutral)  → Balanceado, natural indoor
5500K (Daylight) → Neutro, studio padrao
6500K (Cool)     → Clean, profissional, tech

Prompt keywords:
- "warm golden light" (3000K)
- "neutral daylight" (5000-5500K)
- "cool clean lighting" (6500K)
```

## FUNDOS (BACKGROUNDS)

### Cores Solidas

```
#FFFFFF (White)    → Marketplace requirement
#F5F5F5 (Off-white)→ Softer, less harsh
#000000 (Black)    → Premium, luxury
#E0E0E0 (Gray)     → Neutral, versatile
Colored            → Brand-specific
```

### Texturas

```
Marble      → Luxury, cosmetics, jewelry
Wood        → Organic, natural, artisanal
Concrete    → Industrial, modern, tech
Fabric      → Soft goods, fashion, textiles
Paper       → Clean, minimal, stationery
```

### Gradientes

```
White to light gray   → Subtle depth
Top-lit gradient      → Professional studio feel
Vignette              → Focus on center
```

**Prompt Examples**:
```
"pure white seamless backdrop #FFFFFF"
"rustic weathered wooden surface"
"polished marble background"
"soft gradient from white to light gray"
```

## COMPOSICAO

### Regra dos Tercos

```
Produto posicionado em interseccoes do grid 3x3.
Cria balanco visual dinamico.

Prompt: "rule of thirds composition, product placed at
intersection point"
```

### Centralizado

```
Produto exatamente no centro.
Simetria e foco direto.

Prompt: "centered composition, symmetrical framing,
product in exact center of frame"
```

### Espaco Negativo

```
Area vazia ao redor do produto.
Destaca o sujeito, clean look.

Prompt: "generous negative space, minimalist composition,
product isolated with breathing room"
```

### Leading Lines

```
Linhas guiam o olho para o produto.
Props ou ambiente criam direcao.

Prompt: "leading lines drawing attention to product,
directional composition"
```

### Proporcao Aurea

```
Espiral matematica para posicionamento.
Naturalmente agradavel.

Prompt: "golden ratio composition, fibonacci spiral placement"
```

## PROFUNDIDADE DE CAMPO (DOF)

### Deep DOF (f/8 - f/16)

```
Todo o produto em foco.
Ideal para e-commerce.

Prompt: "deep depth of field, everything in sharp focus,
f/8 aperture, tack sharp throughout"
```

### Shallow DOF (f/1.4 - f/4)

```
Foco seletivo, fundo desfocado.
Lifestyle e emotional shots.

Prompt: "shallow depth of field, creamy bokeh background,
subject in sharp focus, f/2.8 aperture"
```

### Selective Focus

```
Area especifica em foco.
Destaca feature ou detalhe.

Prompt: "selective focus on [specific feature], surrounding
areas gently blurred"
```

## PROPS E STYLING

### Tipos de Props

```
Complementares → Cafe para caneca, plantas para vaso
De escala     → Mao, moeda, objetos conhecidos
Contextuais   → Ambiente de uso (mesa, cozinha)
Decorativos   → Flores, tecidos, elementos visuais
```

### Regras de Styling

1. **Props nao devem competir** com produto
2. **Cores complementares** ao produto
3. **Materiais coerentes** com brand
4. **Quantidade limitada** (2-3 props max)
5. **Proposito claro** (escala, contexto, mood)

- [ ] Fundo branco puro #FFFFFF

## ERROS COMUNS

### 1. Sombras Indesejadas

```
Problema: Sombras no fundo branco
Solucao: Mais luz de fill, produto suspenso, ou
         especificar "no shadows" no prompt
```

### 2. Produto Cortado

```
Problema: Bordas do produto fora do frame
Solucao: "product fully visible", "complete product in frame",
         "nothing cropped"
```

### 3. Foco Incorreto

```
Problema: Parte errada em foco
Solucao: "sharp focus on [specific part]",
         "deep DOF for complete sharpness"
```

### 4. Escala Incerta

```
Problema: Tamanho do produto nao claro
Solucao: Adicionar objeto de referencia,
         "next to [reference] for scale"
```

### 5. Cor Incorreta

```
Problema: Cor do produto diferente da realidade
Solucao: "accurate color representation",
         especificar cor exata no prompt
```
