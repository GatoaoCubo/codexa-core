# Vocabulário de Câmera para Vídeo AI

**Guia Completo** - Termos cinematográficos que as AIs entendem

---

## Movimentos de Câmera

### Dolly (Movimento Linear)
Câmera se move em trilho, aproximando ou afastando.

| Termo | Descrição | Uso |
|-------|-----------|-----|
| `dolly in` | Aproxima do sujeito | Intensificar emoção, revelar detalhe |
| `dolly out` | Afasta do sujeito | Revelar contexto, finalizar cena |
| `dolly forward` | Movimento para frente | Entrada em cena, foco progressivo |
| `dolly back` | Movimento para trás | Revelação, despedida |
| `slow dolly` | Movimento lento | Drama, contemplação |
| `fast dolly` | Movimento rápido | Urgência, impacto |

**Exemplo**:
```
"Slow dolly in toward the glowing product, revealing intricate details"
```

### Pan (Rotação Horizontal)
Câmera gira no eixo vertical, olhando para os lados.

| Termo | Descrição | Uso |
|-------|-----------|-----|
| `pan left` | Gira para esquerda | Revelar elemento à esquerda |
| `pan right` | Gira para direita | Revelar elemento à direita |
| `slow pan` | Panorâmica lenta | Explorar ambiente |
| `whip pan` | Pan muito rápido | Transição energética |

**Exemplo**:
```
"Slow pan right revealing the modern office space and city skyline"
```

### Tilt (Rotação Vertical)
Câmera gira no eixo horizontal, olhando para cima ou baixo.

| Termo | Descrição | Uso |
|-------|-----------|-----|
| `tilt up` | Olha para cima | Revelar altura, grandeza |
| `tilt down` | Olha para baixo | Revelar chão, humildade |
| `slow tilt` | Movimento lento | Drama, revelação |

**Exemplo**:
```
"Tilt up from product to reveal entrepreneur's confident face"
```

### Tracking (Acompanhamento)
Câmera segue o movimento do sujeito.

| Termo | Descrição | Uso |
|-------|-----------|-----|
| `tracking shot` | Segue sujeito lateralmente | Acompanhar caminhada |
| `following shot` | Segue por trás | Jornada, perseguição |
| `lead shot` | Câmera à frente do sujeito | Conversa, confronto |
| `side tracking` | Segue ao lado | Movimento dinâmico |

**Exemplo**:
```
"Tracking shot following the entrepreneur as she walks through the office"
```

### Crane/Jib (Movimento Vertical Amplo)
Câmera sobe ou desce com braço mecânico.

| Termo | Descrição | Uso |
|-------|-----------|-----|
| `crane up` | Sobe revelando | Épico, conclusão |
| `crane down` | Desce para sujeito | Introdução, foco |
| `crane shot` | Movimento vertical amplo | Escala, drama |

**Exemplo**:
```
"Crane up from close-up of product to wide shot of entire studio"
```

### Zoom (Mudança de Focal)
Alteração da lente, não movimento físico.

| Termo | Descrição | Uso |
|-------|-----------|-----|
| `zoom in` | Aproxima visualmente | Foco, urgência |
| `zoom out` | Afasta visualmente | Revelar, contexto |
| `slow zoom` | Gradual | Tensão, antecipação |
| `crash zoom` | Muito rápido | Choque, impacto |

**Exemplo**:
```
"Slow zoom in on the entrepreneur's confident expression"
```

### Orbit (Rotação ao Redor)
Câmera circula o sujeito.

| Termo | Descrição | Uso |
|-------|-----------|-----|
| `orbit` | Circula sujeito | 360° de produto |
| `slow orbit` | Rotação lenta | Exploração detalhada |
| `half orbit` | Meia volta | Revelar outro lado |
| `360 orbit` | Volta completa | Produto showcase |

**Exemplo**:
```
"Smooth 360 orbit around the floating sneaker, revealing all angles"
```

### Handheld (Câmera na Mão)
Movimento orgânico, menos estável.

| Termo | Descrição | Uso |
|-------|-----------|-----|
| `handheld` | Tremor natural | Documentário, urgência |
| `handheld tracking` | Seguindo com tremor | Energia, realismo |
| `steadicam` | Estabilizado mas orgânico | Fluidez cinematográfica |

**Exemplo**:
```
"Handheld tracking shot following the entrepreneur through busy office"
```

### Especiais

| Termo | Descrição | Uso |
|-------|-----------|-----|
| `FPV` | First Person View | Imersão, drone |
| `POV` | Point of View | Perspectiva do personagem |
| `aerial` | Vista aérea | Estabelecer local |
| `static` | Câmera parada | Estabilidade, foco |
| `locked off` | Completamente fixa | Formalidade |

## Ângulos de Câmera

### Verticais

| Termo | Efeito Psicológico | Uso |
|-------|-------------------|-----|
| `eye level` | Neutralidade, igualdade | Conexão, diálogo |
| `low angle` | Poder, grandeza, heroísmo | Hero shots, produtos premium |
| `high angle` | Vulnerabilidade, contexto | Overview, submissão |
| `overhead` | Deus, controle total | Layouts, food, products |
| `worm's eye` | Extremo poder | Monumentos, grandeza extrema |
| `bird's eye` | Overview total | Contexto amplo |

### Horizontais

| Termo | Efeito | Uso |
|-------|--------|-----|
| `front` | Direto, confronto | Apresentação |
| `profile` | Contemplação | Drama |
| `3/4 angle` | Dimensional | Produtos, retratos |
| `over the shoulder` | Perspectiva pessoal | Diálogo |
| `dutch angle` | Tensão, desorientação | Thriller, desconforto |

## Combinações Efetivas

### Para Produtos
```
"Slow orbit, shallow depth of field, macro details"
"Static overhead shot, even lighting"
"Dolly in with rack focus to reveal features"
```

### Para Pessoas
```
"Medium shot, eye level, slight handheld"
"Low angle, slow dolly in, heroic framing"
"Tracking shot, following from side"
```

### Para Ambientes
```
"Establishing wide, crane down to subject"
"Slow pan revealing space"
"FPV flying through environment"
```

## Taxa de Sucesso por Plataforma

### Movimentos Confiáveis (Todas)
- `slow dolly in/out`
- `static shot`
- `slow pan`
- `slow orbit`
- `slow zoom in`

### Movimentos Arriscados
- `whip pan` (pode causar artefatos)
- `fast zoom` (distorção)
- `360 rotation` (inconsistência)
- `complex combined movements`

### Regra de Ouro
```
Um movimento principal por shot
Movimentos lentos = maior qualidade
Especificar direção claramente
```
