## 1. OVERVIEW

### What This Prompt Does

This is a **complete competitive research system** that transforms any LLM into a
specialized Brazilian e-commerce market analyst. When you provide your product
information, the LLM will:

- Analyze competitor positioning and pricing strategies
- Identify market gaps and opportunities
- Extract winning copy patterns and keywords
- Generate actionable recommendations for differentiation
- Create a reusable knowledge document for future use

### The Core Philosophy

**You're not just getting a response. You're building PROPRIETARY KNOWLEDGE.**

Every output from this prompt can be saved and reused across:
- Any LLM (ChatGPT, Claude, Gemini, etc.)
- Any future conversation
- Any team member
- Any time period

This is what we call **"Portable Proprietary Knowledge"** - the new investment.

### What Makes This Different

| Traditional Approach | This Prompt |
|---------------------|-------------|
| Generic competitor search | Brazilian marketplace-specific analysis |
| Surface-level data | Deep strategic insights |
| One-time use | Reusable knowledge asset |
| Model-dependent | Works on any LLM |
| You do the work | AI does the work, you validate |

## 1. RESUMO EXECUTIVO

### Visão Geral do Mercado

O mercado de Whey Protein no Mercado Livre é altamente competitivo, com mais
de 50.000 anúncios ativos. A categoria é dominada por três players principais:
Growth (líder em volume), Integral Médica (líder em percepção de qualidade)
e Max Titanium (melhor custo-benefício percebido). O preço médio para produtos
similares (Whey Isolado 900g) varia entre R$ 130 e R$ 200, com a faixa de
R$ 140-160 apresentando maior conversão.

### Principais Descobertas

1. **Gap de Comunicação**: 78% dos anúncios focam apenas em especificações
   técnicas, negligenciando benefícios emocionais e de estilo de vida

2. **Oportunidade de Preço**: Seu preço de R$ 149,90 está na "zona de ouro"
   - abaixo do premium (Growth R$ 179), acima do genérico (R$ 99)

3. **Diferencial Subutilizado**: Apenas 12% dos concorrentes destacam "zero
   lactose" como benefício principal, apesar de 23% das avaliações negativas
   mencionarem problemas digestivos

4. **Ameaça Identificada**: Growth está agressivamente expandindo com
   promoções de frete grátis e combos, pressionando margens do mercado

5. **Quick Win**: Anúncios com vídeo de preparo do shake têm 3.2x mais
   conversão na categoria - apenas 8% dos vendedores utilizam

### Recomendação Estratégica Principal

Posicione-se como "O Whey para Quem Tem Estômago Sensível" - explorando o
diferencial de zero lactose como benefício principal, não secundário.
Esta abordagem atinge um segmento subatendido (intolerantes à lactose que
praticam musculação) com menor competição direta e maior willingness-to-pay.
Combine com garantia de satisfação de 30 dias para reduzir a barreira de
entrada para consumidores que já tiveram experiências negativas com outros
produtos.

## 10. TROUBLESHOOTING

### Problem: Output is Too Generic

**Cause**: Input was too vague
**Solution**: Add more specific details:
```
Em vez de: "Garrafa de água"
Use: "Garrafa térmica de aço inox 500ml, isolamento a vácuo dupla parede,
tampa com sistema anti-vazamento, cores disponíveis: preto, azul, rosa.
Mantém gelado 24h, quente 12h. Material: aço 304 food-grade."
```

### Problem: Keywords Don't Seem Relevant

**Cause**: LLM may not have current marketplace data
**Solution**: Validate with actual marketplace search:
1. Take the top 5 keywords suggested
2. Search each one in the actual marketplace
3. Report back irrelevant ones:
```
As seguintes keywords não parecem relevantes quando busco no ML: [X, Y, Z]
Por favor, sugira alternativas baseadas em [observação do que realmente aparece]
```

### Problem: Competitor Analysis is Outdated

**Cause**: LLM knowledge cutoff
**Solution**: Provide current data:
```
Para sua informação, os concorrentes atuais no ML para meu produto são:
1. [Competitor 1] - R$ [X] - [observations]
2. [Competitor 2] - R$ [X] - [observations]
3. [Competitor 3] - R$ [X] - [observations]

Por favor, refaça a análise com estes dados atualizados.
```

### Problem: Recommendations Aren't Actionable

**Cause**: Recommendations too strategic, not tactical
**Solution**: Request specific tactics:
```
A recomendação "[X]" está muito abstrata.
Por favor, detalhe:
1. Exatamente o que fazer passo-a-passo
2. Ferramentas ou recursos necessários
3. Como medir se funcionou
4. Tempo estimado para resultados
```

### Problem: Missing Brazilian Market Specifics

**Cause**: LLM defaulting to US/global patterns
**Solution**: Reframe the request:
```
Por favor, refaça considerando especificamente o mercado brasileiro:
- Comportamento do consumidor BR (parcelamento, frete, confiança)
- Plataformas BR (ML, Amazon BR, Shopee BR, Magalu)
- Regulamentações BR (ANVISA, INMETRO, PROCON)
- Cultura de compra BR (reviews, seller reputation)
```
