## Conteudo Freemium

### Os 3 Conceitos Fundamentais

| # | Prompt | Conceito | O Que Ensina |
|---|--------|----------|--------------|
| 01 | [META_PROMPT](./FREEMIUM_01_META_PROMPT.md) | **Delegacao** | Fazer a IA construir prompts estruturados PARA voce |
| 02 | [PLAN](./FREEMIUM_02_PLAN.md) | **Supervisao** | Fazer a IA mostrar o plano ANTES de executar |
| 03 | [IMPLEMENT](./FREEMIUM_03_IMPLEMENT.md) | **Ativos** | Fazer a IA entregar em formato REUTILIZAVEL |

### O Workflow Completo

```
META-PROMPT (Delegacao)
"Construa um prompt para [X]"
  │
  └─> IA cria prompt estruturado
      Voce SALVA como arquivo
        │
        v
PLAN (Supervisao)
"Mostra o plano antes de executar"
  │
  └─> IA mostra plano de execucao
      Voce APROVA ou ajusta
        │
        v
IMPLEMENT (Ativos)
"Entrega em formato reutilizavel"
  │
  └─> IA entrega documento estruturado
      Voce SALVA como ativo

= CONHECIMENTO PROPRIETARIO
  Portatil, reutilizavel, seu pra sempre
```

### Por Que Esses 3?

Esses sao os **atomos** da interacao inteligente com LLMs:

1. **META-PROMPT**: Voce nao escreve prompts, voce DELEGA
2. **PLAN**: Voce nao executa cego, voce SUPERVISIONA
3. **IMPLEMENT**: Voce nao cria descartaveis, voce cria ATIVOS

Qualquer tarefa complexa e uma combinacao desses 3.
