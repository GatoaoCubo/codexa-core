## O CONCEITO

Tres prompts. Infinitas combinacoes. Conhecimento que escala.

```
META-PROMPT --> Constroi prompts estruturados
     |
     v
   PLAN --> Planeja antes de executar
     |
     v
 IMPLEMENT --> Gera ativos reutilizaveis
     |
     v
   ATIVO --> Alimenta novos ciclos...
```

---

## FLUXO BASICO

### Caminho Completo (Tarefas Complexas)

```
[OBJETIVO]
    --> META-PROMPT (gera prompt)
        --> PLAN (cria plano)
            --> IMPLEMENT (executa)
                --> [ATIVO SALVO]
```

**Use quando**: Tarefa nova, complexa, precisa de estrutura

### Caminho Direto (Tarefas Simples)

```
[OBJETIVO]
    --> IMPLEMENT (executa direto)
        --> [ATIVO SALVO]
```

**Use quando**: Tarefa clara, sem ambiguidade

### Caminho de Planejamento (Decisoes Importantes)

```
[OBJETIVO]
    --> PLAN (valida abordagem)
        --> IMPLEMENT (executa)
            --> [ATIVO SALVO]
```

**Use quando**: Quer validar antes de gastar tempo/tokens

## BIBLIOTECA DE ATIVOS (O Cerebro Empresarial)

### Estrutura Recomendada

```
meu_cerebro_empresarial/
|
|-- contextos/           <-- Quem voce e
|   |-- minha_marca.md
|   |-- meu_publico.md
|   |-- meu_tom_de_voz.md
|
|-- templates/           <-- Padroes reutilizaveis
|   |-- template_anuncio.md
|   |-- template_email.md
|   |-- template_post.md
|
|-- pesquisas/           <-- Dados de mercado
|   |-- pesquisa_concorrencia_2025-01.md
|   |-- pesquisa_publico_2025-01.md
|
|-- campanhas/           <-- Execucoes
|   |-- lancamento_produto_x/
|   |-- campanha_natal_2024/
|
|-- prompts/             <-- Prompts construidos
    |-- prompt_copywriting.md
    |-- prompt_analise.md
    |-- prompt_briefing.md
```

### Como Usar a Biblioteca

1. **Nova tarefa?** --> Procure template existente
2. **Template existe** --> Preencha [VARS] + IMPLEMENT
3. **Template nao existe** --> META-PROMPT (cria) + IMPLEMENT (salva)
4. **Tarefa complexa** --> META-PROMPT + PLAN + IMPLEMENT
