# PROMPT PARA NOTEBOOKLM VIDEO | PHOTO AGENT

**Usar com**: NotebookLM Video Generation
**Objetivo**: Video tutorial de 3-5 minutos simulando execucao do agente
**Tom**: Didatico, profissional, direto

---

## PROMPT

```
Crie um video tutorial explicativo sobre o PHOTO AGENT do {{BRAND_URL}}.

CONTEXTO:
O Photo Agent e um diretor de fotografia de IA especializado em e-commerce. Ele gera prompts profissionais para IAs de imagem (Midjourney, DALL-E, Flux, Imagen) que criam fotos de produto de alta qualidade.

ESTRUTURA DO VIDEO (3-5 minutos):

ABERTURA (15 segundos):
- Apresente o Photo Agent como "seu diretor de fotografia de e-commerce"
- Mencione: "9 fotos profissionais a partir de uma frase"

O PROBLEMA (30 segundos):
- Explique a dor: fotos de produto amadoras nao vendem
- Destaque: fundo bagunçado, iluminacao ruim, angulos errados
- O resultado: baixa conversao, percepcao de baixa qualidade
- Transicao: "O Photo Agent resolve isso com direcao de fotografia profissional"

O INPUT (30 segundos):
- Mostre como o usuario inicia: basta digitar uma frase
- Exemplo: "Quero foto de garrafa termica inox preta 500ml"
- Opcao avancada: pode usar dados do Anuncio Agent
- Enfatize: "NAO e uma conversa. E execucao automatica do workflow completo."

O QUE ELE ENTREGA (60 segundos):

1. PROMPT 1 - GRID 3x3 MASTER:
   - 9 cenas em 1 unica imagem
   - Layout grid profissional
   - 500-800 palavras
   - Ideal para visao geral do produto

2. PROMPT 2 - 9 PROMPTS INDIVIDUAIS:
   - 1 prompt por cena
   - 180-300 palavras cada
   - Maior controle criativo
   - Variações de seed para multiplas opcoes

AS 9 CENAS (90 segundos):
Explique cada cena com seu proposito:

1. PACK SHOT - Produto limpo para catalogo (fundo branco #FFFFFF)
2. HERO SHOT - Impacto visual, destaque do produto
3. DETAIL SHOT - Textura, material, acabamento em close
4. SCALE SHOT - Tamanho real, proporcao com objetos conhecidos
5. LIFESTYLE - Pessoa usando o produto no dia a dia
6. IN-CONTEXT - Produto no ambiente de uso real
7. FLAT LAY - Vista superior, composicao artistica
8. GROUP SHOT - Variantes, cores, tamanhos disponiveis
9. CLEAN SHOT - Marketplace compliant (fundo branco #FFFFFF)

Enfatize: "Cenas 1 e 9 SEMPRE com fundo branco puro para compliance de marketplace"

ESPECIFICACOES TECNICAS (30 segundos):
- Cada prompt inclui especificacoes de camera real
- Exemplo: "Canon EOS R5, 85mm f/2.8, ISO 100"
- Lighting setups profissionais: softbox, ring light, rim light
- Composicao: rule of thirds, golden ratio, centered
- "Prompts tecnicos que IAs entendem"

GATILHOS EMOCIONAIS (30 segundos):
- Cada cena tem um gatilho PNL:
  * Pack Shot → Confianca
  * Hero Shot → Aspiracao
  * Lifestyle → Pertencimento
  * Clean Shot → Credibilidade
- "Fotos que vendem emocionalmente"

COMO USAR O OUTPUT (30 segundos):
1. Copie o prompt gerado
2. Abra sua IA de imagem preferida (Midjourney, DALL-E, Flux)
3. Cole o prompt
4. Faca upload da foto real do seu produto (se a IA suportar)
5. Gere a imagem
6. Use {seed:NUMERO} para variacoes consistentes

CONEXAO COM OUTROS AGENTES (20 segundos):
- Recebe contexto do Anuncio Agent (copy alinhada)
- E o terceiro passo do pipeline
- Fluxo: Pesquisa → Anuncio → Foto
- "Fotos alinhadas com seu posicionamento"

CTA (15 segundos):
- "Transforme seus produtos em arte em {{BRAND_URL}}"
- "9 cenas profissionais, qualquer IA de imagem"

TOM E ESTILO:
- Fale como um diretor de fotografia explicando seu processo
- Use linguagem visual: "iluminacao suave", "composicao equilibrada"
- Mostre o valor: fotos profissionais sem estudio
- Seja artistico mas pratico

MENSAGEM CHAVE A REPETIR:
"O Photo Agent transforma qualquer produto em 9 fotos profissionais. Voce digita uma frase, ele gera prompts tecnicos com camera, iluminacao, composicao e gatilhos emocionais. Use em Midjourney, DALL-E ou qualquer IA de imagem. Pack shot, lifestyle, detalhes - tudo pronto para marketplace."

DADOS NUMERICOS A MENCIONAR:
- 9 cenas padronizadas
- 2 prompts de output (Grid + Individuais)
- Cenas 1 e 9 com fundo branco #FFFFFF
- 10-20 minutos de execucao (full) / 3-5 minutos (quick)
- Funciona em 4+ IAs de imagem

NAO FAZER:
- Nao fale como chatbot
- Nao prometa "fotos perfeitas" - prometa direcao profissional
- Nao seja muito tecnico - foque no resultado visual
- Nao compare com contratar fotografo (deixe implicito)
```
