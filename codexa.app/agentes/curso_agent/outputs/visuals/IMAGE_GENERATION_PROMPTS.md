# Image Generation Prompts - M0 Super Freemium

**Version**: 1.0.0 | **Date**: 2025-11-29
**Purpose**: Ready-to-use prompts for AI image generation (Midjourney, DALL-E, Ideogram, etc.)
**Style**: Clean, modern, minimalist infographic style suitable for educational content

## STYLE GUIDE (Apply to All)

### Base Style Prompt
```
Clean minimalist infographic, modern flat design, educational illustration,
white background, professional business style, high contrast,
sharp clean lines, no gradients, geometric shapes,
tech startup aesthetic, 16:9 aspect ratio
```

### Color Palette (Reference)
- Primary Green: #10B981 (success, positive, action)
- Info Blue: #3B82F6 (information, neutral)
- Warning Yellow: #F59E0B (attention, caution)
- Error Red: #EF4444 (problem, avoid)
- Premium Purple: #8B5CF6 (premium, special)
- Dark Text: #1F2937
- Light Background: #F3F4F6

```
Clean minimalist infographic showing "The 4 Cores of AI Agents" concept.

Central element: A glowing brain icon or neural network symbol in the center
representing "AI AGENT" with 4 connected pillars around it.

Four quadrants arranged in a diamond or cross pattern:
1. TOP - "MODEL" (Brain icon) - Blue color #3B82F6 - labeled "The Brain"
2. RIGHT - "CONTEXT" (Book/folder icon) - Green color #10B981 - labeled "Knowledge" - THIS ONE GLOWS BRIGHTER

Connecting lines from each quadrant to the center.

to show it's the differentiator.

educational infographic, no text except labels, geometric icons.
```

## BATCH GENERATION TIPS

### For Midjourney
Add to any prompt:
```
--ar 16:9 --style raw --v 6
```

For cleaner infographics:
```
--ar 16:9 --style raw --v 6 --s 50
```

### For DALL-E 3
Prepend:
```
I NEED to test how the tool works with extremely simple prompts.
DO NOT add any detail, just use it AS-IS:
```

### For Ideogram
Add:
```
Style: Flat design infographic
Text rendering: ON (for any text needed)
```

### For Canva AI
Use the descriptions directly, Canva handles style automatically.

## OUTPUT SPECIFICATIONS

### For Video (10 min)
- Resolution: 1920x1080 (Full HD)
- Format: PNG with transparency where possible
- Naming: `M0_diagram_[name]_v1.png`

### For Landing Page
- Resolution: 2x for retina (3840x2160 for heroes)
- Format: WebP or optimized PNG
- Naming: `landing_[section]_v1.webp`

### For Social Media
- Instagram: 1080x1080
- LinkedIn: 1200x627
- Twitter: 1200x675
