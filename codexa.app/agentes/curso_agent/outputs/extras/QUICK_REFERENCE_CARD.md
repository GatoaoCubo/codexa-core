## FLUXO BÁSICO DE USO

```
1. CARREGA    →  /prime-[agente]
2. DESCREVE   →  Dá contexto do que quer
3. RECEBE     →  Output do agente
4. REFINA     →  Pede ajustes se necessário
5. VALIDA     →  /prime-mentor (opcional)
```

## ESTRUTURA DE PROMPT IDEAL

```
CONTEXTO: [Meu negócio/produto é...]
OBJETIVO: [Quero conseguir...]
ESPECÍFICO: [Preciso exatamente de...]
FORMATO: [Quero em formato de lista/tabela/texto]
```

**Exemplo:**
```
CONTEXTO: Vendo garrafas térmicas no Mercado Livre
OBJETIVO: Aumentar conversão do meu anúncio
ESPECÍFICO: Título de 60 chars + 5 bullets
FORMATO: Pronto pra copiar
```

## QUICK FIXES UNIVERSAIS

| Problema | Solução |
|----------|---------|
| Output genérico | Adicione mais contexto/detalhes |
| Muito longo | "Resuma em [X] palavras/linhas" |
| Muito curto | "Expanda com mais detalhes" |
| Tom errado | "Use tom mais [formal/casual/urgente]" |
| Faltou algo | "Adicione também [X]" |
| Confuso | "Reformula de forma mais clara" |

## CHECKLIST RÁPIDO - ANÚNCIO

- [ ] Título com keyword no início
- [ ] Máx 60 chars (ML) / 80 chars (Shopee)
- [ ] 5 bullet points = benefícios
- [ ] Descrição com PAS ou AIDA
- [ ] Compliance verificado
- [ ] CTA no final

---

## CHECKLIST RÁPIDO - PESQUISA

- [ ] Definiu objetivo claro
- [ ] Escolheu tipo (Quick/Standard/Comprehensive)
- [ ] Analisou 4 pilares (Posição/Preço/Features/Reviews)
- [ ] Identificou gaps
- [ ] Tem recomendação de preço

---

## CHECKLIST RÁPIDO - MARCA

- [ ] Posicionamento escrito
