# NOTAS DE DIREÇÃO

**Para o Apresentador:**
- Ritmo: Pausado mas confiante (não apressado)
- Tom: 60% formal, 80% anti-establishment
- Ênfase: Palavras-chave "Meta-Construção", "Cérebro Plugável", "100%"
- Pausa dramática de 1s em "Você está na camada errada"
- Voz mais firme (não agressiva) em "Cérebro Plugável - potencial: 100%"

**Para o Editor:**
- Cortes sincronizados com animações (não antes/depois)
- Camada 3 deve ter "peso visual" maior (bordas mais grossas, glow, zoom)
- Música cresce sutilmente (+3dB) durante Camada 3
- Split screen final deve aparecer rápido (não fade lento)
- Último overlay "CODEXA TE LEVA PRA CAMADA 3" é o punch line - deixar na tela 3s antes de transição

**Para Motion Designer:**
- Usar easing "ease-out" para reveals (não linear)
- KNOWLEDGE_GLOW não pode ser agressivo (sutil, 30% opacity máximo)
- Ícones de IA devem ser minimalistas (outline 2px, não filled)
- Setas conectoras animadas (desenham-se, não aparecem instant)
