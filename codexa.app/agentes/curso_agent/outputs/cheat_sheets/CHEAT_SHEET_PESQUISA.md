# CHEAT SHEET: Pesquisa Agent

**10 Prompts Prontos para Copiar e Usar**

---

## SETUP

```
/prime-pesquisa
```

---

## 1. QUICK RESEARCH (15-20 min)

```
Quick research: [produto/categoria] no [marketplace]

Quero saber:
- Tem demanda? (volume estimado)
- Top 5 concorrentes
- Faixa de preço (min/med/max)
- Nível de concorrência (baixo/médio/alto)
- Vale a pena entrar? (sim/não + por quê)
```

---

## 2. STANDARD RESEARCH (30-40 min)

```
Standard research: [produto/categoria]

PILAR 1 - Posicionamento:
- Como os top 5 se posicionam?
- Qual diferencial de cada?

PILAR 2 - Preços:
- Faixa dominante
- Quem cobra premium e por quê?

PILAR 3 - Features:
- O que TODO MUNDO oferece?
- O que NINGUÉM oferece? (gaps)

PILAR 4 - Reviews:
- Principais elogios
- Principais reclamações
```

---

## 3. ANÁLISE DE CONCORRENTE ESPECÍFICO

```
Analisa o concorrente [nome/link]:

Quero saber:
1. Posicionamento (premium/econômico/nicho)
2. Estratégia de preço
3. Pontos fortes (copiar)
4. Pontos fracos (explorar)
5. Como me diferenciar dele
```

---

## 4. GAP ANALYSIS (Oportunidades)

```
Identifica gaps de mercado em [categoria]:

Analisa:
- O que clientes PEDEM nos reviews mas não encontram
- Features que ninguém oferece
- Faixas de preço mal atendidas
- Públicos ignorados
- Problemas não resolvidos

Me dá pelo menos 5 oportunidades concretas.
```

---

## 5. ESTRATÉGIA DE PREÇO

```
Define estratégia de preço para:

MEU PRODUTO:
- Descrição: [produto]
- Custo: R$ [X]
- Margem mínima desejada: [X]x

CONCORRÊNCIA:
- Preço mais baixo: R$ [?]
- Preço médio: R$ [?]
- Preço mais alto: R$ [?]

Recomenda preço e justifica:
- É premium, médio ou econômico?
- Que argumentos uso pra justificar?
- Posso cobrar mais se [condição]?
```

---

## 6. VALIDAÇÃO DE IDEIA

```
Valida essa ideia de produto:

PRODUTO: [descreva]
INVESTIMENTO: R$ [valor]
MARGEM ESPERADA: [X]%

Analisa:
1. Tem demanda suficiente?
2. Concorrência é superável?
3. Margem é realista?
4. Riscos principais?
5. Recomendação: entra ou não? Por quê?
```

---

## 7. COMPARAÇÃO DE MARKETPLACES

```
Compara [produto] em diferentes marketplaces:

Analisa: ML, Shopee, Amazon BR

Para cada um:
- Volume de demanda
- Preço médio
- Nível de concorrência
- Taxas e custos
- Onde vale mais a pena vender?
```

---

## 8. ANÁLISE DE TENDÊNCIAS

```
Analisa tendências para [categoria]:

Quero saber:
1. Está crescendo ou caindo? (YoY)
2. Sazonalidade (meses fortes/fracos)
3. Produtos em alta na categoria
4. Produtos em declínio
5. Previsão para próximos 6 meses
```

---

## 9. ANÁLISE DE REVIEWS

```
Analisa reviews de [produto/categoria]:

Coleta dos top 5 concorrentes:
- O que clientes AMAM (manter)
- O que clientes ODEIAM (evitar/resolver)
- Pedidos frequentes (oportunidades)
- Palavras mais usadas (usar no copy)

Formato: tabela comparativa
```

---

## 10. COMPREHENSIVE RESEARCH (60+ min)

```
Comprehensive research: [categoria] para entrada no mercado

Seção 1: MERCADO
- Tamanho e crescimento
- Players principais
- Tendências

Seção 2: CONCORRÊNCIA
- Análise dos top 10
- Posicionamento de cada
- Gaps identificados

Seção 3: CONSUMIDOR
- Perfil do comprador
- Jornada de compra
- Dores e desejos

Seção 4: OPORTUNIDADE
- Nicho recomendado
- Diferenciação possível
- Estratégia de entrada

Seção 5: FINANCEIRO
- Faixa de preço ideal
- Margem esperada
- Investment needed
```

---

## QUICK FIXES

| Preciso de... | Comando Rápido |
|---------------|----------------|
| Só preços | "Lista preços dos top 10 em tabela" |
| Só keywords | "Keywords mais buscadas nessa categoria" |
| Só gaps | "3 oportunidades que ninguém explora" |
| Atualização | "Dados mais recentes disponíveis" |
| Validação rápida | "Em 1 parágrafo: vale ou não vale?" |

## DICAS PRO

```
Para pesquisa mais precisa:
- Especifique marketplace: "no ML" / "na Shopee"
- Especifique região: "Brasil" / "São Paulo"
- Especifique período: "últimos 3 meses"
- Especifique faixa: "R$ 50-150"
```
