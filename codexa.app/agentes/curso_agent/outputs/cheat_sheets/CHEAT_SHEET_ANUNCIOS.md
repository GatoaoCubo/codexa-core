# CHEAT SHEET: Anuncio Agent

**10 Prompts Prontos para Copiar e Usar**

---

## SETUP

```
/prime-anuncio
```

---

## 1. ANÚNCIO COMPLETO (Mais usado)

```
Cria anúncio completo para [MARKETPLACE]:

PRODUTO:
- Nome: [nome do produto]
- Categoria: [categoria]
- Preço: R$ [preço]
- Diferencial: [o que torna único]

QUERO:
1. Título otimizado (máx [60/80] caracteres)
2. 5 bullet points focados em benefícios
3. Descrição de [300-500] palavras
4. Validação de compliance
```

---

## 2. TÍTULO OTIMIZADO SEO

```
Cria 5 versões de título para Mercado Livre:

Produto: [nome]
Keywords obrigatórias: [lista]
Limite: 60 caracteres

Ordena do mais ao menos otimizado para SEO.
```

---

## 3. BULLET POINTS PERSUASIVOS

```
Cria 5 bullet points para [produto]:

FOCO em BENEFÍCIOS, não características.
Use números quando possível.
Inclua pelo menos 1 prova social ou garantia.

Formato:
✓ [benefício] - [como entrega]
```

---

## 4. DESCRIÇÃO COM STORYTELLING

```
Escreve descrição de [X] palavras usando PAS:

PROBLEMA: [dor do cliente]
AGITAÇÃO: [por que é urgente]
SOLUÇÃO: [como produto resolve]

Produto: [detalhes]
Tom: [casual/formal/urgente]
Inclui CTA no final.
```

---

## 5. ADAPTAR PARA OUTRO MARKETPLACE

```
Adapta esse anúncio de [origem] para [destino]:

[cole o anúncio atual]

Considera:
- Limite de caracteres do título: [X]
- Tom preferido: [descreva]
- O que converte mais nessa plataforma
```

---

## 6. CHECKLIST DE COMPLIANCE

```
Verifica compliance para [produto]:

Preciso saber:
1. Requer registro ANVISA? Qual tipo?
2. Requer selo INMETRO? Qual categoria?
3. Avisos obrigatórios no anúncio?
4. Restrições de claims (não pode dizer)?
5. Checklist completo antes de publicar
```

---

## 7. VARIAÇÕES A/B

```
Cria 3 variações A/B para testar:

Anúncio base:
[cole título e bullets atuais]

Variação A: Foco em [benefício 1]
Variação B: Foco em [benefício 2]
Variação C: Foco em [urgência/escassez]

Mantém mesmo produto, muda ângulo de venda.
```

---

## 8. KEYWORDS RESEARCH

```
Lista keywords para [produto] no [marketplace]:

Organiza em:
- 5 keywords principais (alto volume)
- 5 keywords de intenção de compra
- 5 long-tail (alta conversão)

Formato:
[keyword] - volume estimado - dificuldade
```

---

## 9. OTIMIZAR ANÚNCIO EXISTENTE

```
Analisa e otimiza esse anúncio:

[cole anúncio atual completo]

Me diz:
1. O que está bom (manter)
2. O que está ruim (mudar)
3. O que está faltando (adicionar)
4. Versão otimizada completa
```

---

## 10. FICHA TÉCNICA FORMATADA

```
Formata ficha técnica para [produto]:

Dados brutos:
[cole especificações]

Quero:
- Tabela organizada por categoria
- Unidades de medida padronizadas
- Termos traduzidos (se em inglês)
- Formato compatível com [marketplace]
```

---

## QUICK FIXES

| Problema | Comando Rápido |
|----------|----------------|
| Título longo | "Encurta para máx 60 chars mantendo keywords" |
| Copy fraco | "Adiciona gatilhos: escassez + prova social" |
| Sem CTA | "Adiciona CTA urgente no final" |
| Genérico | "Torna mais específico pro público [X]" |
| Sem benefícios | "Transforma características em benefícios" |

---

## ATALHOS DE FORMATO

```
Para output limpo, adicione no final:

"Formato: só o resultado, sem explicações"
"Formato: markdown pronto pra copiar"
"Formato: lista numerada"
"Formato: tabela"
```
