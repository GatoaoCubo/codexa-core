# CHEAT SHEET: Marca Agent

**10 Prompts Prontos para Copiar e Usar**

---

## SETUP

```
/prime-marca
```

---

## 1. DESCOBRIR ARQUÉTIPO

```
Descobre meu arquétipo de marca:

NEGÓCIO: [descreva em 1 linha]
PÚBLICO: [quem são seus clientes]
VALORES: [3-5 valores importantes]
REFERÊNCIAS: [marcas que admira]

Me dá:
- Arquétipo PRIMÁRIO (80%)
- Arquétipo SECUNDÁRIO (20%)
- Por que essa combinação
- Como isso se manifesta na comunicação
```

---

## 2. CRIAR POSICIONAMENTO

```
Cria posicionamento usando o framework:

"Para [público] que [necessidade],
[marca] é [categoria] que [benefício].
Diferente de [concorrência], nós [diferencial]."

MEU NEGÓCIO:
- Produto/Serviço: [X]
- Público: [X]
- Diferencial: [X]
- Concorrentes: [X]
```

---

## 3. DEFINIR TOM DE VOZ

```
Define tom de voz da minha marca:

Marca é: [arquétipo escolhido]
Público é: [descreva]

Quero:
1. Posição 1-10 nas 4 dimensões:
   - Formal ↔ Casual
   - Sério ↔ Divertido
   - Respeitoso ↔ Irreverente
   - Técnico ↔ Simples

2. Lista de 10 palavras que USO
3. Lista de 10 palavras que EVITO
4. 3 exemplos de frases no tom certo
```

---

## 4. CRIAR PALETA DE CORES

```
Cria paleta de cores para minha marca:

Arquétipo: [X]
Sensação desejada: [X]
Referências visuais: [marcas/estilos]

Quero:
- 2-3 cores primárias (com hex)
- 1-2 cores secundárias (com hex)
- 1 cor de destaque/CTA (com hex)
- Justificativa psicológica de cada
```

---

## 5. ESCOLHER TIPOGRAFIA

```
Recomenda tipografia para minha marca:

Arquétipo: [X]
Tom de voz: [formal/casual/etc]
Uso principal: [web/impresso/ambos]

Quero:
- Fonte para títulos (e por quê)
- Fonte para corpo (e por quê)
- Fonte alternativa se não encontrar
- Links para download (Google Fonts)
```

---

## 6. BRAND GUIDELINES COMPLETO

```
Gera Brand Guidelines completo:

MARCA: [nome]
ARQUÉTIPO: [X]
PÚBLICO: [X]

Inclui:
1. Essência da marca (1 parágrafo)
2. Posicionamento
3. Tom de voz (4 dimensões)
4. Paleta de cores (com hex)
5. Tipografia
6. 5 DO's
7. 5 DON'Ts
8. Exemplos de aplicação
```

---

## 7. NAMING (Sugestão de Nome)

```
Sugere nomes para minha marca:

NEGÓCIO: [descreva]
VALORES: [X]
TOM: [X]
EVITAR: [palavras/estilos que não quer]

Quero:
- 10 opções de nome
- Significado/origem de cada
- Disponibilidade provável de domínio .com.br
- Top 3 recomendados e por quê
```

---

## 8. TAGLINE/SLOGAN

```
Cria tagline para minha marca:

MARCA: [nome]
POSICIONAMENTO: [1 frase]
DIFERENCIAL: [X]

Quero:
- 5 opções de tagline
- Máximo 6 palavras cada
- Memorável e diferenciador
- Indica qual funciona melhor e por quê
```

---

## 9. HISTÓRIA DA MARCA (Brand Story)

```
Escreve a história da minha marca:

FUNDADOR: [você]
ORIGEM: [como começou]
PROBLEMA: [que percebeu]
SOLUÇÃO: [que criou]
MISSÃO: [por que existe]

Formato:
- Versão curta (50 palavras) - bio de rede social
- Versão média (150 palavras) - sobre nós
- Versão longa (300 palavras) - página institucional
```

---

## 10. VALIDAR CONSISTÊNCIA

```
Valida consistência da minha marca:

ARQUÉTIPO: [X]
POSICIONAMENTO: [X]
TOM DE VOZ: [X]
CORES: [X]
TIPOGRAFIA: [X]

Analisa:
1. Elementos conflitam entre si?
2. Brand Consistency Score (0-1)
3. O que ajustar para melhorar?
4. Está coerente para o público [X]?
```
