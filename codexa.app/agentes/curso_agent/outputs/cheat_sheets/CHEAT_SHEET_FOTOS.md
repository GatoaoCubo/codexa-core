# CHEAT SHEET: Photo Agent

**10 Prompts Prontos para Copiar e Usar**

---

## SETUP

```
/prime-photo
```

---

## 1. GRID 3x3 COMPLETO (Mais usado)

```
Cria grid 3x3 de fotos para [produto]:

REQUISITOS [MARKETPLACE]:
- Resolução: [1200x1200 / 1000x1000]
- Fundo: [branco / lifestyle]

ESTILO: [Commercial / Minimalist / Lifestyle]
MARCA: [nome] - cores: [hex]

CENAS:
1. Hero shot - fundo branco
2. Detalhe de material/textura
3. Vista superior
4. Lifestyle em uso
5. Comparação de escala
6. Embalagem
7. Contexto específico [academia/casa/etc]
8. Detalhe funcional
9. Infográfico de features

Gera prompts otimizados para cada.
```

---

## 2. HERO SHOT PROFISSIONAL

```
Cria prompt para hero shot de [produto]:

REQUISITOS:
- Fundo branco puro (#FFFFFF)
- Produto ocupa 85% do frame
- Iluminação softbox profissional
- Sem sombras duras
- Resolução: 2000x2000px

ESTILO: Commercial catalog
CÂMERA: Canon EOS R5, 100mm macro
```

---

## 3. FOTO LIFESTYLE

```
Cria prompt para foto lifestyle:

PRODUTO: [X]
CONTEXTO: [onde está sendo usado]
PESSOA: [descreva se tiver]

ESTILO:
- Iluminação natural / golden hour
- Profundidade de campo (bokeh)
- Ambiente realista, não estúdio
- Emoção: [alegria/calma/energia/etc]
```

---

## 4. FOTO DE DETALHE/TEXTURA

```
Cria prompt para close-up de detalhe:

PRODUTO: [X]
FOCO: [qual parte/material]

TÉCNICA:
- Macro lens (100mm)
- f/2.8 para bokeh
- Textura visível (poros, fibras, metal)
- Iluminação lateral para relevo
```

---

## 5. COMPARAÇÃO DE ESCALA

```
Cria prompt para mostrar tamanho:

PRODUTO: [X]
TAMANHO REAL: [X cm/ml/etc]

COMPARAR COM:
- Mão humana
- Objeto comum [caneta/celular/moeda]
- [outro produto da linha]

Deixar proporção clara sem parecer amador.
```

---

## 6. FOTO DE EMBALAGEM

```
Cria prompt para foto de embalagem:

PRODUTO: [X]
TIPO EMBALAGEM: [caixa/sacola/envelope]
MARCA: [nome + cores]

MOSTRAR:
- Embalagem fechada (premium feel)
- Parcialmente aberta (reveal)
- Unboxing experience
- Materiais sustentáveis se aplicável
```

---

## 7. INFOGRÁFICO COM FEATURES

```
Cria prompt para foto infográfica:

PRODUTO: [X] no centro

FEATURES PARA DESTACAR:
1. [Feature 1] - ícone: [sugestão]
2. [Feature 2] - ícone: [sugestão]
3. [Feature 3] - ícone: [sugestão]
4. [Feature 4] - ícone: [sugestão]

ESTILO: Clean, setas minimalistas
CORES: [paleta da marca]
```

---

## 8. MÚLTIPLOS ÂNGULOS

```
Cria prompts para 4 ângulos do mesmo produto:

PRODUTO: [X]

ÂNGULOS:
1. Frontal (0°)
2. 3/4 (45°)
3. Lateral (90°)
4. Traseira (180°)

Manter:
- Mesma iluminação
- Mesmo fundo
- Mesma distância
- Consistência total
```

---

## 9. ADAPTAR PARA MARKETPLACE

```
Adapta essas fotos para [marketplace]:

FOTOS ATUAIS: [descreva ou cole prompts]

REQUISITOS [MARKETPLACE]:
- Mercado Livre: 1200x1200, branco, 85% preenchido
- Amazon: 1000x1000, branco puro, sem texto
- Shopee: 800x800, pode lifestyle, texto ok

O que preciso ajustar em cada prompt?
```

---

## 10. VARIAÇÕES A/B

```
Cria 3 variações da mesma foto para teste:

FOTO BASE: [descreva]

VARIAÇÃO A: Fundo branco clássico
VARIAÇÃO B: Fundo lifestyle contextual
VARIAÇÃO C: Fundo com cor da marca

Qual tende a converter mais para [público]?
```

## PARÂMETROS TÉCNICOS

```
Adicione ao final do prompt:

CÂMERA: Canon EOS R5 / Sony A7IV / Nikon Z8
LENTE: 50mm f/1.4 (geral) / 85mm f/1.8 (retrato) / 100mm macro (detalhe)
ILUMINAÇÃO: Softbox / Natural / Ring light / Dramatic
ASPECT RATIO: --ar 1:1 (square) / --ar 4:5 (portrait) / --ar 16:9 (banner)
QUALIDADE: --quality 2 / --hd / --upscale
```

## QUICK FIXES

| Problema | Comando |
|----------|---------|
| Foto genérica | "Adiciona elemento único: [gota água/reflexo/textura]" |
| Cores erradas | "Ajusta cenário pra paleta [hex codes]" |
| Muito escuro | "Iluminação mais clara, estilo Apple Store" |
| Sem vida | "Adiciona elemento humano ou movimento" |
| Stock photo | "Mais autêntico, menos perfeito" |
