# CHEAT SHEET: Mentor Agent

**10 Prompts Prontos para Copiar e Usar**

---

## SETUP

```
/prime-mentor
```

---

## 1. TIRAR DÚVIDA RÁPIDA

```
Me explica [conceito/termo] de forma simples.

Quero entender:
- O que é
- Por que importa pro meu negócio
- Exemplo prático
- Próximo passo
```

---

## 2. PEDIR AULA COMPLETA

```
Me ensina sobre [tópico]:

Formato de aula:
1. Por que isso importa? (motivação)
2. Os 3-5 pilares essenciais (conceitos)
3. Como fazer passo-a-passo (prática)
4. Exemplo real de seller brasileiro
5. Exercício pra eu fazer agora
```

---

## 3. VALIDAR MEU TRABALHO

```
Valida esse [anúncio/pesquisa/marca]:

[Cole seu trabalho aqui]

Me diz:
1. Score de 0-10
2. O que está bom (manter)
3. O que está ruim (mudar)
4. O que está faltando (adicionar)
5. Versão melhorada se score < 8
```

---

## 4. QUAL AGENTE USAR?

```
Quero fazer [descreva a tarefa].

Qual agente devo usar?
- /prime-anuncio?
- /prime-pesquisa?
- /prime-marca?
- /prime-photo?
- /prime-codexa?

Explica por que e como começar.
```

---

## 5. RESOLVER PROBLEMA

```
Estou tendo problema:

SITUAÇÃO: [descreva]
O QUE TENTEI: [o que você fez]
O QUE ACONTECEU: [resultado]
O QUE ESPERAVA: [resultado desejado]

Me ajuda a resolver.
```

---

## 6. COMPARAR OPÇÕES

```
Tenho duas opções:

OPÇÃO A: [descreva]
OPÇÃO B: [descreva]

CONTEXTO: [sua situação]
OBJETIVO: [o que quer alcançar]

Qual escolher? Por quê?
```

---

## 7. CRIAR PLANO DE AÇÃO

```
Quero [objetivo]:

MEU CONTEXTO:
- Negócio: [X]
- Experiência: [iniciante/intermediário/avançado]
- Tempo disponível: [X] horas/semana
- Prioridade: [velocidade/qualidade/custo]

Cria plano de ação com:
1. Passos em ordem
2. Tempo estimado de cada
3. Qual agente usar em cada passo
4. Como saber que completei
```

---

## 8. PROCESSAR MEU CONHECIMENTO

```
Processa esse conteúdo para minha base:

[Cole conteúdo: pode ser texto, transcrição, anotações]

Extrai:
- Resumo executivo (2-3 parágrafos)
- Conceitos-chave (lista)
- Como aplicar no meu negócio
- Tags para catalogar
```

---

## 9. EXPLICAR COMO ESPECIALISTA

```
Sou [perfil] e quero entender [tópico].

Meu nível: [iniciante/intermediário/avançado]
Meu contexto: [tipo de negócio]

Explica como se fosse um mentor experiente,
não um professor acadêmico.
Use exemplos do mercado brasileiro.
```

---

## 10. FEEDBACK CONSTRUTIVO

```
Fiz isso: [descreva ou cole]

Objetivo era: [o que queria alcançar]

Me dá feedback:
1. Funcionou? Por quê (não)?
2. O que eu faria diferente?
3. Pontuação de 0-10
4. 1 coisa pra melhorar primeiro
```

---

## TOM DO MENTOR

O Mentor fala como:
- ✅ "Olha só, funciona assim..."
- ✅ "Isso aqui já vi dar certo em 100+ lojas..."
- ✅ "No ML, SEO é basicamente título + keywords"

Não como:
- ❌ "Conforme a literatura acadêmica..."
- ❌ "Implementar estratégia multifacetada..."
- ❌ "De acordo com metodologias..."

---

## QUICK FIXES

| Preciso de... | Comando |
|---------------|---------|
| Resposta curta | "Responde em 1 parágrafo" |
| Resposta longa | "Explica em detalhes, pode ser extenso" |
| Mais exemplos | "Me dá 3 exemplos práticos" |
| Sem teoria | "Só prática, sem teoria" |
| Passo a passo | "Formato: lista numerada de ações" |

## ESTRUTURA DE PERGUNTA IDEAL

```
CONTEXTO: [Meu negócio é X, vendo Y]
SITUAÇÃO: [O que está acontecendo]
OBJETIVO: [O que quero alcançar]
PERGUNTA: [O que preciso saber/fazer]
```

**Exemplo:**
```
CONTEXTO: Tenho loja de acessórios femininos no ML
SITUAÇÃO: Conversão caiu 30% no último mês
OBJETIVO: Voltar ao patamar anterior
PERGUNTA: Por onde começo a investigar?
```
