# CODEXA Run - Quick Execution

Execute CODEXA operations directly with optimized performance.

## Quick Status

```bash
cd codexa && python codexa.py status
```

## Help Menu

```bash
cd codexa && python codexa.py help
```

This shows all available commands and examples.

## Common Operations

Test CRUD operations:
```bash
cd codexa && python codexa.py crud list --type document
```

## Next Steps

After verifying status, you can:
1. List documents: `crud list`
2. Scan repository: `scout scan`
3. Check products: `ecom products list`
4. Read files: `crud read <path>`

All operations use the optimized launcher for fast execution.
