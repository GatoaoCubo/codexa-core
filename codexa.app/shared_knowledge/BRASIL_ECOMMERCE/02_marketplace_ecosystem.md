## Platform Deep Dives

### Mercado Livre

**Position**: Dominant leader, Latin America's largest

**Key Metrics**:
- Monthly visitors: 350M+
- Mobile app: #1 shopping app BR
- Fulfillment: Mercado Envios (MELI logistics)
- Fintech: Mercado Pago integration

**Seller Requirements**:
```
Account Types:
- Pessoa Fisica (CPF): Limited categories
- Pessoa Juridica (CNPJ): Full access

Fees (2024):
- Commission: 11-16% (category dependent)
- Mercado Envios: Variable by weight/distance
- Ads: CPC model (Mercado Ads)

Reputation Tiers:
- Verde (Green): Full benefits
- Amarelo (Yellow): Reduced visibility
- Vermelho (Red): Restrictions apply
```

**Listing Optimization**:
| Element | Best Practice |
|---------|---------------|
| Title | 60 chars, main keyword first |
| Photos | 1200x1200px, white BG, 6+ images |
| Description | Structured, bullet points |
| Ficha Tecnica | 100% attributes filled |
| Price | Competitive, consider installments |

**Fulfillment Options**:
1. **Full**: MELI warehouse, 24h delivery
2. **Flex**: Seller ships, MELI last-mile
3. **Coleta**: MELI pickup from seller
4. **Proprio**: Seller handles all logistics

### Amazon BR

**Position**: Fast growth, premium positioning

**Key Metrics**:
- Prime members: 8M+ (estimated)
- FBA coverage: 90%+ of Brazil
- Categories: 30+ departments

**Seller Requirements**:
```
Account Types:
- Individual: R$ 2/sale (< 10 items/month)
- Professional: R$ 19/month + commission

Fees (2024):
- Referral: 8-15% (category)
- FBA: Weight + dimensions formula
- Storage: Monthly cubic meter rate

Performance Metrics:
- Order Defect Rate: < 1%
- Late Shipment: < 4%
- Cancellation: < 2.5%
```

**Amazon-Specific Features**:
- A+ Content (Brand Registry required)
- Amazon Ads (Sponsored Products, Brands, Display)
- Subscribe & Save
- Amazon Vine (reviews program)

**Image Requirements**:
```
Main Image:
- Pure white background (RGB 255,255,255)
- Product fills 85%+ of frame
- No text, logos, watermarks
- 1600px minimum (zoom enabled)

Additional Images:
- Lifestyle shots allowed
- Infographics permitted
- 360-degree view bonus
```

## Cross-Border Considerations

### Remessa Conforme (2024)

New regulation for international e-commerce:

```
Compliant Platforms:
- Declared value accurate
- Pre-paid import tax (ICMS 17%)
- Simplified customs

Tax-Free Threshold:
- US$ 50 (was US$ 0 - changed 2024)
- Applies to compliant platforms only

Non-Compliant:
- Full taxation at border
- Delays, additional fees
```

**Affected Platforms**: Shein, AliExpress, Shopee (cross-border), Temu

## Multi-Channel Strategy

### Recommended Approach

```
Phase 1: Single Platform
- Start with highest-fit marketplace
- Master operations, build reviews
- Typical: Mercado Livre or Shopee

Phase 2: Dual Platform
- Add complementary marketplace
- Use integration hub
- Typical: + Amazon BR or Magalu

Phase 3: Omnichannel
- 3-5 marketplaces
- Own e-commerce (Nuvemshop, VTEX)
- Social commerce (Instagram, WhatsApp)
```
