## Overview

Brazilian Portuguese (PT-BR) copywriting differs significantly from European Portuguese and requires cultural adaptation beyond translation. This guide covers linguistic patterns, regional nuances, and persuasion frameworks specific to Brazilian e-commerce.

---

## Core Linguistic Principles

### PT-BR vs PT-PT Key Differences

| Aspect | PT-BR | PT-PT |
|--------|-------|-------|
| Gerund | Estou comprando | Estou a comprar |
| You (formal) | Voce | Tu/Voce |
| Placement | Proclitic (me da) | Enclitic (da-me) |
| Vocabulary | Celular, trem, onibus | Telemovel, comboio, autocarro |

**Rule**: ALWAYS use PT-BR patterns. PT-PT reads as foreign/formal to Brazilians.

### Tone Spectrum

```
Formal ←————————————————————→ Informal
       |         |         |
    Banking   E-commerce  Social

Recommended for e-commerce: Center-right (friendly professional)
```

**Avoid**:
- Overly formal: "Prezado cliente, informamos que..."
- Overly casual: "E ai, mano! Bora comprar?"

**Target**: "Oi! Encontrou o que procurava? A gente ta aqui pra ajudar."

## Persuasion Frameworks (BR)

### PASTOR (Adapted for BR)

```
P - Problema: Identify pain point
A - Amplificacao: Intensify consequences
S - Solucao: Present product as answer
T - Transformacao: Show before/after
O - Oferta: Clear value proposition
R - Resposta: Strong CTA
```

### Brazilian-Specific Triggers

| Trigger | Why It Works | Example |
|---------|--------------|---------|
| Parcelamento | Installment culture | "12x sem juros" |
| Frete Gratis | High shipping costs BR | "Frete gratis acima de R$ 99" |
| Pronta Entrega | Impatience culture | "Envio imediato" |
| Garantia | Trust deficit | "30 dias de garantia" |
| Avaliacoes | Social proof obsession | "4.8 estrelas (2.340 avaliacoes)" |
| Exclusividade | Status seeking | "Edicao limitada" |

### Trust Signals (Critical for BR)

```markdown
Essential:
[ ] CNPJ visible
[ ] Physical address
[ ] WhatsApp contact
[ ] Clear return policy
[ ] Real customer reviews

Bonus:
[ ] Reclame Aqui rating
[ ] RA1000 seal
[ ] Ebit reputation
[ ] SSL certificate badge
[ ] Payment method logos
```

## Title Formulas

### Marketplace Titles (60-80 chars)

**Formula**: [Product] + [Key Attribute] + [Benefit/Differentiator]

```
Good:
"Fone Bluetooth 5.3 Cancelamento de Ruido 40h Bateria"
"Panela de Pressao Eletrica 6L Inox 12 Funcoes 110V"
"Camiseta Dry Fit Masculina Academia Secagem Rapida"

Bad:
"FONE DE OUVIDO SUPER MEGA POTENTE IMPERDIVEL!!!!"
"Panela - Compre Agora - Melhor Preco"
"Camiseta"
```

### SEO Title Patterns

| Category | Pattern |
|----------|---------|
| Electronics | [Brand] [Model] [Key Spec] [Connectivity] |
| Fashion | [Item] [Gender] [Material] [Style] [Occasion] |
| Home | [Product] [Material] [Capacity] [Voltage] |
| Beauty | [Product] [Benefit] [Skin/Hair Type] [Size] |

## Description Structures

### Bullet Point Formula (Benefits-First)

```markdown
Por que escolher [produto]?

✓ [Benefit 1]: [Feature that enables it]
✓ [Benefit 2]: [Feature that enables it]
✓ [Benefit 3]: [Feature that enables it]
✓ [Benefit 4]: [Feature that enables it]

Especificacoes:
• [Spec 1]
• [Spec 2]
• [Spec 3]

O que vem na caixa:
• 1x [Product]
• 1x [Accessory]
• 1x [Manual]
```

### Long-Form Description (Storytelling)

```markdown
[Hook - Problem/Desire]
Cansado de [pain point]? Voce nao esta sozinho.

[Agitate]
[Describe frustration, consequences]

[Solution]
Apresentamos [product]: [one-sentence value prop]

[Features as Benefits]
[Bullet list with benefit-first approach]

[Social Proof]
Mais de [number] clientes satisfeitos...

[Guarantee/Risk Reversal]
Se nao gostar, devolvemos seu dinheiro.

[CTA]
Garanta o seu agora - estoque limitado!
```

## Common Mistakes to Avoid

### Linguistic Errors

| Error | Wrong | Correct |
|-------|-------|---------|
| Literal translation | "Livre gratis" | "Frete gratis" |
| PT-PT vocabulary | "Telemovel" | "Celular" |
| Formal excess | "Vossa Senhoria" | "Voce" |
| Anglicism overuse | "Fazer o check-out" | "Finalizar compra" |
| Gender mismatch | "A smartphone" | "O smartphone" |

### Cultural Missteps

| Mistake | Why It Fails | Fix |
|---------|--------------|-----|
| Cold, impersonal tone | Brazilians value warmth | Add friendly touches |
| No installment option | 70% prefer parcelado | Always show "12x" |
| US-centric references | Different cultural context | Localize examples |
| Ignoring WhatsApp | Primary communication channel | Offer WA support |
| No Reclame Aqui presence | Trust verification habit | Maintain good rating |

```
Start: November 1st (Black November)
Peak: Last Friday of November
Extension: Cyber Monday (less relevant in BR)

Copy Elements:

- "Menor preco do ano"
- Comparison with regular price
- Stock counters

```
