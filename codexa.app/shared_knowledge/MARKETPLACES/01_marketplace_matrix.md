## DETAILED COMPARISON

### 1. MERCADO LIVRE (ML)

**Profile**:
- Founded: 1999 (Argentina), Brazil operations dominant
- Seller Types: Individual, Professional, Loja Oficial
- Fulfillment: Mercado Envios, Full, Flex
- Payment: Mercado Pago (integrated)

**Strengths**:
- Largest reach in LATAM
- Strong logistics network (Full = 1-2 day delivery)
- Integrated payment ecosystem
- Best for high-volume sellers

**Weaknesses**:
- High competition, price pressure
- Strict listing policies
- Fees: 11-17% (varies by category)

**Best For**:
- Electronics, auto parts, home goods
- Established brands
- High-volume sellers
- Products with clear differentiators

**Seller Tiers**:
| Tier | Requirements | Benefits |
|------|--------------|----------|
| MercadoLider | Sales + Reputation | Featured placement, lower fees |
| Loja Oficial | Brand verification | Brand store, premium support |

---

### 2. SHOPEE BRASIL

**Profile**:
- Founded: 2015 (Singapore), Brazil entry 2019
- Seller Types: Individual, Shopee Mall
- Fulfillment: Shopee Xpress, partner carriers
- Payment: ShopeePay, credit cards, PIX

**Strengths**:
- Aggressive marketing (campaigns, vouchers)
- Mobile-first UX (90%+ mobile users)
- Lower entry barrier for new sellers
- Strong in fashion, beauty, accessories

**Weaknesses**:
- Lower average order value
- Race to bottom on pricing
- Less brand credibility vs ML/Amazon

**Best For**:
- Fashion, beauty, accessories
- New sellers testing market
- Products under R$100
- Impulse purchases

**Key Features**:
| Feature | Description |
|---------|-------------|
| Flash Sales | High visibility, time-limited offers |
| Shopee Live | Live commerce integration |
| Shopee Ads | In-platform advertising |
| Cupons | Heavy voucher/coupon culture |

---

### 3. AMAZON BRASIL

**Profile**:
- Founded: 2012 (Brazil marketplace), full expansion 2019
- Seller Types: Individual, Professional, Brand Registry
- Fulfillment: FBA (Fulfillment by Amazon)
- Payment: Amazon Pay, credit cards

**Strengths**:
- Global brand trust
- Prime ecosystem (loyalty)
- Best A+ Content options
- Strong in books, electronics, imported goods

**Weaknesses**:
- Smaller market share vs ML
- Stricter listing requirements
- Higher fulfillment costs

**Best For**:
- Brand-registered products
- Premium/imported goods
- Products benefiting from A+ Content
- International expansion potential

**Unique Features**:
| Feature | Description |
|---------|-------------|
| A+ Content | Enhanced brand content (images, comparison tables) |
| Brand Registry | IP protection, brand store |
| Subscribe & Save | Recurring purchase discounts |
| Amazon Vine | Review program for new products |

---

### 4. MAGAZINE LUIZA (MAGALU)

**Profile**:
- Founded: 1957 (traditional retail), marketplace 2016
- Seller Types: 3P Sellers, Magalu Empresas
- Fulfillment: Magalu Entregas, partner carriers
- Payment: MagaluPay, credit cards

**Strengths**:
- Strong omnichannel (1000+ physical stores)
- Trust from traditional retail history
- Good for electronics, appliances
- Regional strength (interior Brasil)

**Weaknesses**:
- Stricter product requirements (EAN mandatory)
- Smaller digital-native audience
- Less aggressive marketing vs Shopee

**Best For**:
- Electronics, appliances
- Products needing physical store support
- Regional expansion (interior)
- B2B sales (Magalu Empresas)

**Requirements**:
| Requirement | Status |
|-------------|--------|
| EAN Code | Mandatory |
| NCM Code | Mandatory |
| CNPJ | Mandatory |
| Warranty | Required for electronics |

## STRATEGY RECOMMENDATIONS

### By Product Category

| Category | Primary Platform | Secondary | Rationale |
|----------|-----------------|-----------|-----------|
| Electronics | ML | Amazon | Volume + trust |
| Fashion | Shopee | ML | Mobile audience, impulse |
| Beauty/Cosmetics | Shopee | ML | Young demographic |
| Books | Amazon | ML | Category strength |
| Auto Parts | ML | - | Established category |
| Home Appliances | Magalu | ML | Omnichannel + warranty |
| Imported/Premium | Amazon | ML | Brand trust |

### By Seller Profile

| Profile | Recommended Start | Scale To |
|---------|-------------------|----------|
| New Seller, Low Budget | Shopee | ML |
| Established Brand | ML + Amazon | All 4 |
| Imported Goods | Amazon | ML |
| Regional Business | Magalu | ML |
| High Volume | ML (Full) | Amazon (FBA) |

### Multi-Platform Strategy

**Phase 1 - Entry (Months 1-3)**:
- Choose 1 primary platform based on category
- Master listing optimization
- Build reputation/reviews

**Phase 2 - Expansion (Months 4-6)**:
- Add secondary platform
- Adapt content per platform specs
- Unify inventory management

**Phase 3 - Scale (Months 7-12)**:
- Expand to all relevant platforms
- Implement cross-platform analytics
- Optimize fulfillment strategy

## DECISION MATRIX

Use this matrix to select platform priority:

```
IF product_price < R$50 AND category IN (fashion, beauty, accessories)
   → Primary: Shopee | Secondary: ML

IF product_price > R$200 AND category IN (electronics, tech)
   → Primary: ML | Secondary: Amazon

IF brand_registered == true AND product_type == "premium"
   → Primary: Amazon | Secondary: ML

IF requires_physical_support == true OR region == "interior"
   → Primary: Magalu | Secondary: ML

DEFAULT
   → Primary: ML | Secondary: Based on category
```
