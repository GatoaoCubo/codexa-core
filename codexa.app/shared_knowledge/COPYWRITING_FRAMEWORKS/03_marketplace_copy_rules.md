## OVERVIEW

Each marketplace has unique requirements for copy. This card consolidates guidelines for the major Brazilian platforms.

**Target Platforms**: Mercado Livre, Shopee, Amazon BR, Magalu

## MERCADO LIVRE

### Titulo (Title)

**Limits**:
- Minimo: 20 caracteres
- Maximo: 60 caracteres
- Ideal: 58-60 caracteres (maximiza visibilidade)

**Structure Pattern**:
```
[PRODUTO] + [ATRIBUTO PRINCIPAL] + [DIFERENCIAL] + [MODELO/COR]
```

**Examples**:
```
OK:  "Aspirador De Po Vertical 1200w Filtro Hepa Silencioso Preto"
OK:  "Cadeira Gamer Ergonomica Reclinavel 180 Graus Com Apoio"
BAD: "O MELHOR ASPIRADOR DO MERCADO!!!!! SUPER PROMOCAO!!!!"
BAD: "Aspirador" (muito curto)
```

**Palavras Proibidas**:
- MAIUSCULAS EXCESSIVAS
- Simbolos repetidos (!!!, ???, ***)
- "Promocao", "Oferta", "Desconto" no titulo
- "Gratis", "Brinde"
- Marcas que nao sejam do produto

### Bullet Points

**Limits**:
- Quantidade: Ate 10 bullets
- Caracteres: 250-299 por bullet (ideal)
- Minimo recomendado: 200 caracteres

**Structure Pattern**:
```
[BENEFICIO] + [FEATURE] + [PROVA/DIFERENCIAL]
```

**Example**:
```
"Durma melhor com a espuma viscoelastica D45 que se adapta ao
contorno do seu corpo, distribuindo o peso uniformemente para
eliminar pontos de pressao. Certificacao INMETRO garante qualidade
e seguranca para toda a familia. Mais de 5.000 avaliacoes 5 estrelas."
```

**Compliance**:
- Nao repetir informacoes do titulo
- Cada bullet = 1 tema principal
- Incluir especificacoes tecnicas
- Mencionar garantia/certificacoes

### Descricao

**Limits**:
- Minimo efetivo: 3.300 caracteres
- Ideal: 3.500-4.500 caracteres
- Maximo: Sem limite rigido

**Required Sections**:
```
1. Abertura (gancho + relevancia)
2. Beneficios principais
3. Especificacoes tecnicas
4. Modo de uso (se aplicavel)
5. O que esta incluso
6. Garantia e suporte
7. Fechamento (CTA implicito)
```

**Formatting Rules**:
- Pararagrafos curtos (3-4 linhas)
- Listas para specs tecnicas
- Nao usar HTML/markdown
- Espacamento para legibilidade

### Palavras Proibidas (Geral ML)

| Categoria | Palavras |
|-----------|----------|
| Promocional | Promocao, desconto, oferta, liquidacao |
| Urgencia | Corra, ultimo, aproveite agora |
| Garantias falsas | Melhor do mercado, numero 1, garantido |
| Contato | WhatsApp, telefone, email, site |
| Condicoes | Frete gratis, parcelamento (usar campos proprios) |

## SHOPEE

### Titulo

**Limits**:
- Minimo: 25 caracteres
- Maximo: 100 caracteres
- Ideal: 80-100 caracteres

**Structure Pattern**:
```
[PRODUTO] [ATRIBUTOS] [BENEFICIO] [VARIACAO] [QUANTIDADE]
```

**Examples**:
```
OK:  "Kit 3 Camisetas Basicas Masculinas Algodao Premium Anti Odor
      Manga Curta Gola V Preto Branco Cinza"
OK:  "Relogio Inteligente Smartwatch Monitor Cardiaco Pressao
      Bluetooth 5.0 Prova Dagua IP68 Preto"
BAD: "Camiseta" (muito curto)
BAD: "PROMOCAO CAMISETA BARATA MELHOR PRECO" (spam)
```

**Keywords Strategy**:
- Usar sinonimos no titulo
- "Masculino" E "Homem" E "Para Homens"
- Variacoes de escrita (anti-odor, antiodor)

### Descricao

**Limits**:
- Minimo: 100 caracteres
- Ideal: 1.000-2.000 caracteres
- Maximo: 3.000 caracteres

**Structure Pattern**:
```
[DESTAQUE DO PRODUTO]
[ESPECIFICACOES TECNICAS]
[MODO DE USO]
[CONTEUDO DA EMBALAGEM]
[POLITICA DE TROCA]
```

**Shopee-Specific Tips**:
- Usar emojis com moderacao (1-2 por secao)
- Bullet points com simbolos (- ou *)
- Destacar "Frete Gratis" se aplicavel
- Mencionar "Loja Oficial" se for

### Palavras Permitidas (diferente do ML)

- "Promocao" (permitido com moderacao)
- "Frete Gratis" (incentivado)
- "Menor Preco" (com cuidado)

### Palavras Proibidas

- Contatos externos (WhatsApp, Instagram)
- Links para outros sites
- Mencao a concorrentes
- Palavras ofensivas

## AMAZON BR

### Titulo

**Limits**:
- Maximo: 200 caracteres
- Ideal: 150-180 caracteres (mais longo que ML)

**Structure Pattern (Amazon Style)**:
```
[MARCA] [LINHA] [PRODUTO] [ATRIBUTOS] [QUANTIDADE] [COR/TAMANHO]
```

**Examples**:
```
OK:  "Mondial Aspirador de Po Vertical AP-45 1200W com Filtro
      HEPA Lavavel Cabo 5 Metros Preto e Vermelho"
OK:  "Loreal Paris Elseve Shampoo Oleo Extraordinario Nutricao
      Intensa para Cabelos Secos 400ml"
```

**Amazon Title Rules**:
- Marca obrigatoria no inicio
- Nao usar simbolos (!, @, #, $)
- Nao usar "melhor", "mais vendido"
- Capitalizar cada palavra principal

### Bullet Points (Amazon)

**Limits**:
- Quantidade: 5 bullets principais
- Caracteres: 500 por bullet (maximo)
- Ideal: 200-300 caracteres

**Amazon Bullet Pattern**:
```
[KEYWORD]: [Beneficio detalhado com especificacao]
```

**Examples**:
```
"FILTRAGEM HEPA: O filtro de alta eficiencia captura 99.97%
das particulas microscopicas incluindo acaros, polen e pelos
de animais, ideal para pessoas com alergias respiratorias."

"DESIGN ERGONOMICO: Cabo telescopico ajustavel de 80cm a
120cm permite limpar em pe sem dores nas costas, pesando
apenas 2.3kg para facil manuseio em todos os comodos."
```

### A+ Content (Conteudo A+)

**Disponivel para**: Brand Registry owners

**Modules recomendados**:
1. Hero image com beneficios
2. Comparativo com outros modelos da marca
3. Tabela de especificacoes
4. Imagens de uso/lifestyle
5. FAQ visual

**Copy para A+**:
- Textos mais curtos (captions)
- Foco visual
- Bullets simplificados
- CTA implicito

### Backend Keywords (Termos de Busca)

**Limits**:
- 250 bytes total
- Separar com espacos (nao virgulas)
- Nao repetir palavras do titulo

**Strategy**:
```
[sinonimos] [erros comuns] [termos relacionados] [uso]

Exemplo para aspirador:
"aspirar limpar limpeza po poeira sujeira casa apartamento
quarto sala vertical portatil silencioso potente barato"
```

## MAGALU (Magazine Luiza)

### Titulo

**Limits**:
- Maximo: 150 caracteres
- Ideal: 80-120 caracteres

**Structure Pattern**:
```
[PRODUTO] [MARCA] [MODELO] [ATRIBUTOS] [COR]
```

**Example**:
```
"Smartphone Samsung Galaxy A54 5G 128GB 6GB RAM Tela 6.4
Super AMOLED Camera 50MP Preto"
```

### Descricao

**Limits**:
- Sem limite rigido
- Ideal: 1.500-2.500 caracteres

**Magalu-Specific**:
- Suporta HTML basico
- Permite tabelas
- Formatacao visual valorizada

### Ficha Tecnica

**Obrigatoria e detalhada**:
- Preencher TODOS os campos disponiveis
- Afeta ranking de busca
- Quanto mais completo, melhor

## COMPLIANCE CROSS-PLATFORM

### Sempre Evitar

```
- Claims nao comprovanveis ("o melhor", "numero 1")
- Informacoes de contato direto
- Redirecionamento para outros canais
- Precos ou condicoes no texto (usar campos)
- Conteudo copiado de concorrentes
- Fotos com marca d'agua de terceiros
```

### Sempre Incluir

```
- Especificacoes tecnicas precisas
- Garantia e politica de troca
- Conteudo da embalagem
- Modo de uso quando aplicavel
- Certificacoes (INMETRO, ANVISA)
```

### ANVISA/INMETRO

**Produtos regulados**:
- Cosmeticos: Registro ANVISA
- Eletronicos: Selo INMETRO
- Suplementos: Notificacao ANVISA
- Brinquedos: Certificacao INMETRO

**Copy obrigatoria**:
```
"Produto registrado na ANVISA sob numero [X]"
"Certificado INMETRO [NUMERO]"
```
