## OVERVIEW

Identidade visual abrange paletas de cores, tipografia, estilo de imagem e diretrizes visuais que criam reconhecimento de marca e conexao emocional com clientes.

**Por que Identidade Visual Importa**:
- 90% dos julgamentos de produto sao baseados apenas em cor
- Visuais consistentes aumentam reconhecimento de marca em 80%
- Cores influenciam decisoes de compra (85% citam cor como razao primaria)
- Tipografia comunica personalidade antes das palavras serem lidas

---

## PSICOLOGIA DAS CORES

### Processo de Selecao de Cor Primaria

1. **Comece com arquetipo** - Cores que combinam com personalidade
2. **Considere categoria** - O que concorrentes usam? (diferenciar ou alinhar?)
3. **Teste resposta emocional** - Que sentimento a cor evoca?
4. **Verifique marketplace** - Destaca-se nos resultados de busca?

### Estrutura de Paleta de Cores

**Tamanho Recomendado**: 3-5 cores

**Estrutura**:
1. **Primaria** (60% uso): Cor principal da marca, logo, titulos
2. **Secundaria** (30% uso): Destaque, botoes CTA, realces
3. **Neutra** (10% uso): Fundos, texto, bordas

**Opcional**:
4. **Acento** (5% uso): Elementos especiais, badges, alertas
5. **Complementar** (variacoes): Versoes claras/escuras de primaria/secundaria

---

**Exemplo de Paleta** (Marca fitness e-commerce):

```
PRIMARIA: #E63946 (Vermelho Vibrante)
SECUNDARIA: #212529 (Carvao Profundo)
NEUTRA: #F1FAEE (Off-White)
ACENTO: #FFB703 (Amarelo Energetico)

Uso:
- Logo: Vermelho + Carvao
- Botoes CTA: Fundo vermelho + Texto branco
- Titulos: Texto carvao em Off-White
- Badges de Promocao: Acento amarelo
```

## DIRETRIZES DE TIPOGRAFIA

### Estrategia de Pareamento de Fontes

**Sistema de 2 Fontes** (recomendado para e-commerce):
- **Primaria (Titulos)**: Display/sans-serif para impacto
- **Secundaria (Corpo)**: Sans-serif para legibilidade

**Sistema de 3 Fontes** (opcional para marcas premium):
- **Primaria (Headlines)**: Display/serif para elegancia
- **Secundaria (Subtitulos)**: Sans-serif para clareza
- **Terciaria (Corpo)**: Sans-serif para legibilidade

---

### Psicologia de Fontes

| Estilo | Personalidade | Melhor Para | Preferencia BR |
|--------|---------------|-------------|----------------|
| **Serif** (Times, Georgia) | Tradicional, confiavel, elegante | Luxo, juridico, financas | Apenas contextos formais |
| **Sans-Serif** (Arial, Roboto) | Moderno, limpo, acessivel | Tech, varejo, maioria | Preferido (95% dos casos) |
| **Display** (Impact, Bebas) | Bold, chamativo | Esportes, promocoes, jovem | Alto impacto headlines |
| **Script** (Pacifico, Dancing) | Elegante, pessoal, feminino | Beleza, casamentos, artesanal | Use com moderacao (acentos) |
| **Monospace** (Courier, Mono) | Tecnico, moderno, preciso | Tech, codigo, dados | Marcas nicho/tech |

### Escala Tipografica

**Hierarquia** (sizing responsivo):

```
H1 (Hero Headline): 48-72px desktop, 32-40px mobile
H2 (Section Headers): 36-48px desktop, 24-32px mobile
H3 (Subsections): 24-32px desktop, 20-24px mobile
Body (Paragrafos): 16-18px desktop, 14-16px mobile
Small (Legendas): 12-14px desktop, 11-13px mobile
```

**Hierarquia de Peso**:
- Headlines: Bold (700) ou ExtraBold (800)
- Subtitulos: SemiBold (600)
- Corpo: Regular (400)
- Legendas: Regular (400) ou Light (300)

## DIRETRIZES DE ESTILO DE IMAGEM

### Selecao de Estilo de Fotografia

**Escolha UM estilo consistente**:

**1. Lifestyle** (produtos em uso)
- Melhor para: Moda, casa, bem-estar
- Mood: Aspiracional, relacionavel
- Exemplo: Modelo usando produto em ambiente natural

**2. Produto-Focado** (limpo, minimo)
- Melhor para: Tech, acessorios, ferramentas
- Mood: Profissional, claro
- Exemplo: Fundo branco, produto centralizado

**3. Flat Lay** (arranjo de cima)
- Melhor para: Beleza, papelaria, comida
- Mood: Organizado, estetico
- Exemplo: Produtos arranjados em superficie texturizada

**4. Acao** (produto em movimento)
- Melhor para: Esportes, outdoor, gear ativo
- Mood: Dinamico, energetico
- Exemplo: Produto sendo usado durante atividade

**5. Detalhe** (close-up, textura)
- Melhor para: Luxo, artesanal, premium
- Mood: Qualidade, artesanato
- Exemplo: Foto macro de material/textura

---

### Melhores Praticas de Imagem E-commerce BR

**Faca**:
- Use modelos diversos (Brasil e multicultural)
- Mostre escala (adicione referencia de tamanho)
- Iluminacao natural (sensacao quente, tropical)
- Fundos coloridos (nem sempre branco)
- Cenarios da vida real (contextos relacionaveis)

**Evite**:
- Fotos stock (obviamente encenadas/genericas)
- Iluminacao fria/clinica (parece distante)
- Apenas um tom de pele (nao representativo)
- Muito editado (irreal)
- Apenas contextos EUA/Europa (nao relacionavel)

---

### Consideracoes Especificas por Marketplace

**Mercado Livre**:
- Branding amarelo - Evite produtos amarelo puro (mistura)
- Use alto contraste para visibilidade
- Concorrentes azuis dominam - Diferencie com vermelho/laranja

**Shopee**:
- Branding laranja - Evite laranja puro
- Flash sale = vermelho/rosa - Use cores complementares
- Alta competicao - Cores bold performam melhor

**Magazine Luiza**:
- Branding azul - Produtos azuis combinam bem
- Fundos brancos limpos preferidos
- Fotografia profissional sobre criativa

## CHECKLIST DE CONSISTENCIA VISUAL

### Uso do Logo

- [ ] Tamanho minimo: 100px largura (digital), 1.5cm (impresso)
- [ ] Espaco limpo: Minimo altura do logo em todos os lados
- [ ] Variacoes: Cor completa, branco, preto (3 versoes minimo)
- [ ] Formatos: SVG (escalavel), PNG (transparente), JPG (fundos)

### Consistencia de Cor

- [ ] Todas as cores tem codigos HEX + RGB + CMYK documentados
- [ ] Maximo 5 cores na paleta primaria
- [ ] Regra 60-30-10 aplicada (primaria-secundaria-acento)
- [ ] Acessibilidade: Contraste de texto >= 4.5:1 (WCAG AA)

### Consistencia Tipografica

- [ ] Maximo 2-3 familias de fonte
- [ ] Todos os tamanhos/pesos documentados
- [ ] Hierarquia clara (H1 > H2 > H3 > Body)
- [ ] Altura de linha: 1.5-1.8 para texto corpo

### Consistencia de Imagem

- [ ] Estilo de fotografia definido (1-2 estilos primarios)
- [ ] Dimensoes de imagem padronizadas
- [ ] Estilo de edicao consistente (filtros, brilho, saturacao)
- [ ] Contextos e modelos focados no Brasil

## ACESSIBILIDADE (Conformidade WCAG)

### Requisitos de Contraste de Cor

**WCAG AA** (padrao minimo):
- Texto normal: Ratio de contraste 4.5:1
- Texto grande (18px+): Ratio de contraste 3:1

**WCAG AAA** (aprimorado):
- Texto normal: Ratio de contraste 7:1
- Texto grande: Ratio de contraste 4.5:1

**Ferramentas de Teste**:
- WebAIM Contrast Checker
- Color Safe (gera paletas acessiveis)
- Stark (plugin Figma)

---

### Consideracoes de Daltonismo

**Tipos afetando 8% dos homens, 0.5% das mulheres**:
- Deuteranopia (verde-cego): Mais comum
- Protanopia (vermelho-cego)
- Tritanopia (azul-cego)

**Solucoes**:
- Nao dependa apenas de cor (adicione icones, texto, padroes)
- Use paletas seguras para daltonicos (azul + laranja, roxo + amarelo)
- Teste com simuladores (Coblis, Color Oracle)

## ENTREGAVEIS DE IDENTIDADE VISUAL

**Pacote visual completo da marca inclui**:

1. **Guia de Paleta de Cores** (PDF)
   - Cores primaria/secundaria/neutra
   - Codigos HEX, RGB, CMYK
   - Diretrizes de uso
   - Classificacoes de acessibilidade

2. **Guia de Tipografia** (PDF)
   - Familias de fonte + pesos
   - Hierarquia de tamanhos
   - Exemplos de pareamento
   - Links de web fonts (Google Fonts)

3. **Arquivos de Logo** (ZIP)
   - Cor completa (SVG, PNG, JPG)
   - Versao branca (SVG, PNG)
   - Versao preta (SVG, PNG)
   - Layouts horizontal + vertical
   - Tamanhos minimos documentados

4. **Diretrizes de Imagem** (PDF)
   - Exemplos de estilo de fotografia
   - O que fazer e nao fazer
   - Presets de edicao
   - Requisitos de diversidade de modelos

5. **Brand Board** (JPG/PDF)
   - Referencia visual de uma pagina
   - Cores + fontes + logo + estilo de imagem
   - Referencia rapida para designers
