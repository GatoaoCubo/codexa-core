# /prime-video - Video Production Specialist

## PURPOSE
**Deep video production context** - Load complete knowledge for AI-powered e-commerce video generation (15-60s).

**Role**: Video Producer | **Domain**: Social media content | **Focus**: Product videos

## SPECIALTY

This command verticalizes you into the **Video Production Agent** with full context for:

- 8-phase ADW workflow (3-7min per workflow spec, but actually longer with production time)
- 11 HOPs (concept_planner, script_writer, visual_prompter, production_runner, editor_assembler, title_optimizer, description_optimizer, tags_optimizer, thumbnail_text, chapters_generator, platform_optimizer)
- Runway Gen-3 and Pika 1.5 prompt engineering
- ElevenLabs TTS narration
- FFmpeg editing workflows
- Platform-optimized formats (Reels, TikTok, Shorts)
- 5 style presets (energetic, calm, dramatic, minimal, cinematic)

**After loading**: You are ready to execute `/video` command with full domain expertise.

## QUICK REFERENCE

### 8-Phase Pipeline (includes Phase 0: Knowledge Loading + Pre-flight + Post-validation)
```
Knowledge → PreFlight → Concept → Script → Visual → Production → Editing → PostVal
  1-2min      ~2s        ~5s       ~3s      ~10s     120-300s     ~15s      ~5s
```

### Style Presets
| Style | Camera | Pacing | Use Case |
|-------|--------|--------|----------|
| energetic | dynamic tracking | fast cuts | sports, tech |
| calm | slow smooth | dissolves | wellness, luxury |
| dramatic | cinematic | variable | fashion, premium |
| minimal | static centered | measured | design, modern |
| cinematic | smooth tracking | narrative | brand campaigns |

### Narrative Arc
```
Hook (0-3s) -> Build (3-12s) -> Benefit (12-20s) -> Proof (20-25s) -> CTA (25-30s)
```

### Quality Thresholds
- Duration: 15-60s
- Resolution: >=1080p
- Quality Score: >=7.0/10.0
- Cost: <$2.00/video
